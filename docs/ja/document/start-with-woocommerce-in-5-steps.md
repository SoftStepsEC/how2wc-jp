この入門ガイドでは、WooCommerce を使用して Web サイトを設定する方法を説明します。

推奨事項については、[公式 WooCommerce ホスティング](https://woocommerce.com/hosting-solutions/) ページを参照してください。

WordPress.com 以外のホスティングサービスを選ぶ場合は、無料の WordPress ソフトウェア プラットフォームを利用できます。多くのホスティング会社ではワンクリックで WordPress をインストールできるオプションを提供しているので、このオプションがあるか確認するか、利用可能かどうか問い合わせてください。または、[WordPress.org](https://wordpress.org/download/) からダウンロードし、[WordPress のインストール方法](https://wordpress.org/support/article/how-to-install-wordpress/) の手順に従ってインストールすることもできます。

### WooCommerce を有効にする

セルフホスト型のウェブサイトが稼働したら、次は**WooCommerce**を使ってeコマース機能をインストールしましょう。インストール方法は3つあります。

- WooCommerce.com で無料アカウントを作成
- WordPress.com と WooCommerce.com でアカウントを登録
- 以下から選択：
- 信頼できるホストにプリインストールされたソリューション
- 既に WordPress サイトをお持ちの場合は WooCommerce を自動インストール
- zip ファイルをダウンロードして自分でインストール
    
  
- WooCommerce をインストール
- WooCommerce プラグインをダウンロード
- プラグインをダウンロード
- ウェブサイトへアクセス
- ダッシュボード > プラグイン > 新規追加 > アップロード
- ダウンロードしたプラグインを選択
- インストール
- 有効化
    
- ウェブサイト内からWooCommerceプラグインを検索
- ウェブサイトへ移動
- **ダッシュボード > プラグイン > 新規追加 > プラグインの検索** へ移動
- 「WooCommerce」を検索
- インストール
- 有効化
    
  

プラグインを有効化すると、[セットアップウィザード](https://woocommerce.com/document/woocommerce-setup-wizard/)が表示されます。初期設定をスムーズに行うため、以下の手順に従うことを強くお勧めします。

上記の**セットアップウィザード**では、テーマを選択するよう求められます。選択したテーマによって、ストア/サイトの初期の外観とレイアウトが決まります（ただし、そこからカスタマイズすることも可能です）。WordPressに付属の無料テーマを使用することも、プレミアムテーマを購入することもできます。

WooCommerce向けに構築・最適化されている**Storefront**とその**子テーマ**をおすすめします。Storefrontは無料です。機能を追加したい場合は、**Storefront拡張機能**をご購入いただけます。

Storefront があなたに合わない場合は、WooCommerce に最適な他の**テーマ**が多数あります。

**拡張機能**で機能を追加して、ストアを成長させましょう。[サブスクリプション](https://woocommerce.com/products/woocommerce-subscriptions/)、[予約](https://woocommerce.com/products/woocommerce-bookings/)、[メンバーシップ](https://woocommerce.com/products/woocommerce-memberships/)、[決済ゲートウェイ](https://woocommerce.com/product-category/woocommerce-extensions/payment-gateways)、[配送](https://woocommerce.com/product-category/woocommerce-extensions/shipping-methods)など、ほぼすべての拡張機能を販売しています。

無料でストア/ウェブサイトを立ち上げ、顧客基盤と収益の拡大に合わせてスケールアップできます。各拡張機能は、サポートと製品アップデートにアクセスできるサブスクリプションでご購入いただけます。サブスクリプションは年間請求となります。

次の手順については、[WooCommerce ドキュメント](https://woocommerce.com/documentation/woocommerce/getting-started/settings/)をご覧ください。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
