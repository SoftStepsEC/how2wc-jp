title: 商品のカスタムフィールド

WooCommerce は、カスタム フィールドを使用して製品リストを強化する機能を提供します。この機能により、追加の製品詳細を顧客に表示できるようになります。

カスタムフィールドは、商品の追加の[メタデータ](https://developer.wordpress.org/plugins/metadata/)であり、テーマやサイト内でさまざまな方法で利用して、追加情報を表示したり、ショッピングエクスペリエンスをカスタマイズしたり、デフォルトの商品フィールドに含まれていない特定の商品の詳細を提供したりできます。

この柔軟性により、オンラインストアの機能を目標に合わせて拡張できます。このガイドでは、WordPress管理画面からWooCommerce製品にカスタムフィールドを直接追加し、サイトに表示する手順を詳しく説明します。

カスタムフィールドについて詳しく学ぶ前に、カスタムフィールドと商品属性の違いを理解することが重要です。主な違いは、その用途と機能にあります。

- **商品属性** は、幅広い商品を分類するための基本要素として機能します。[タクソノミー](https://developer.wordpress.org/themes/basics/categories-tags-custom-taxonomies/) と同様に機能し、顧客はこれらの属性に基づいて商品を絞り込んだりグループ化したりできます。この機能は、商品間に意味のある関係性を構築するために不可欠であり、色、サイズ、素材などの属性を通じて顧客が商品を探し、見つけやすくします。
- 一方、**カスタムフィールド** は個々の商品に合わせてカスタマイズされ、その商品に固有の情報やオプションを提供します。商品をより広いグループに分類する属性とは異なり、カスタムフィールドは商品ごとに固有の詳細情報やパーソナライズの選択肢を提供します。

本質的には、属性は共通の特性に基づいて製品をカテゴリまたはグループに整理する役割を果たしますが、カスタム フィールドは特定のアイテムに固有の追加情報を提供することで、製品の個々のリストに深みを加えます。

製品のカスタム フィールドを有効にするには、次の手順に従ってください。

1. **WP-admin > 商品** へ移動します。
2. 「新規追加」ボタンを使用して新しい商品を追加するか、既存の商品を「編集」します。
3. 商品編集ページで、画面右上の **「画面オプション」** をクリックします。
4. ドロップダウンから **「カスタムフィールド」オプションを見つけてチェックを入れ、商品に対して有効にします。**

製品のカスタム フィールドを有効にしたら、製品にカスタム データを追加できるようになります。

1. 同じ商品編集ページで、**一番下にある「カスタムフィールド」セクションまでスクロールダウンしてください**
2. 新しいカスタムフィールドを追加するには、**「カスタムフィールドを追加」ボタン**をクリックして新しいフィールドを作成します
3. **「名前」**入力ボックスにカスタムフィールドの名前を入力します。この名前は、テーマまたはサイト内でプログラム的にフィールドを参照する際に使用されます
4. **「値」**入力ボックスにカスタムフィールドの値を入力します。追加の仕様、一意の識別子、カスタムメモなど、商品に関連するあらゆる情報を指定できます
5. カスタムフィールドを追加し、対応する値を入力したら、商品を保存または更新してください

各商品のカスタムフィールドを表示するには、テーマのファイルを編集する必要があります。その方法について詳しくは、[WooCommerce 開発者ドキュメント](https://developer.woocommerce.com/docs/displaying-custom-fields-in-your-theme-or-site/)をご覧ください。
