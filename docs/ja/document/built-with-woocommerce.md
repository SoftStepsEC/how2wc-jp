何かをオンラインで注文し、商品が発送されるというメールを受け取ると、ワクワクします。

しかし、注文した商品が届かなかったらどうすればいいでしょうか？あるいは、返品した商品の返金を期待しているのでしょうか？気が変わって注文をキャンセルしたい場合はどうすればいいでしょうか？

WooCommerce.com は、オープンソースソフトウェアを使用しているサイトやストアにアクセスできず、決済プロセスも監視できないため、注文を履行できなかった販売者に対して措置を講じることはできません。しかし、あなたは可能です！

以下では、まず何が起こったのかを説明し、次に次のステップの提案を示します。

WooCommerce ストアから送信される注文メールのフッターに「WooCommerce で構築」と表示されます (下のスクリーンショットの #1 を参照)。

古い W​​ooCommerce ストアから送信された注文メールのフッターに「Powered by WooCommerce」と表示されます。

一部のオンライン ストアのフッターにも「WooCommerce で構築」と表示されます。

これらの情報から、WooCommerce に問い合わせるべきだと推測する人が多いようです。しかし、WooCommerce は実際にはサイト所有者がオンラインストアを運営するために使用している無料のオープンソースソフトウェアです。

WooCommerceチームはストア、支払い、注文の履行を管理しません。それらはサイト所有者の責任となります。

注文確認メールに記載されている「My Awesome Shop」は、WooCommerce ソフトウェアを使用して商品を販売しているショップの名前です (上記のスクリーンショットの #2)。

ウェブサイトの URL またはサイト アドレスは「https://myawesomeshop.local」(上記のスクリーンショットの #3) です。

商品を配送してもらう、または返金してもらうために取ることができる行動についての提案をまとめました。

受信トレイに届く注文確認メールは、ウェブサイトの管理者/オーナーとして登録されているメールアドレスから送信されます。このメールアドレスを使用して、ストアオーナーに直接返信してください。

返答がない場合は、ストアのサイト（上記スクリーンショットの＃3）にアクセスし、サイトを通じてストアのオーナーに直接連絡してください。

ほとんどのサイトには、連絡先ページ、ヘルプ オプション、またはソーシャル メディア アカウントがあります。

販売者とサイトを銀行またはクレジットカード会社に報告し、請求に異議を申し立てます（例では、「https://myawesomeshop.local」の「My Awesome Shop」になります）。

銀行やクレジットカード会社には通常、「請求への異議申し立て方法」などのページがあります。必要に応じて、不正利用の調査が開始される場合があります。

サイトのホスティング会社またはドメインレジストラに問い合わせてみてください。WooCommerceソフトウェアを動作させるには、ホスティング会社との連携が必要です。

この解決策は、前の 2 つの手順を実行して、自分で解決しようとした明確な記録を確立し、これらの企業が役立つ可能性があることをケースに追加することで、成功する可能性が高くなります。

- **ホスティング会社**: サイトのデータをサーバー上に保管している会社です。ホスティングプロバイダーを確認するには、[WhoisHostingthis.com](https://www.whoishostingthis.com/) にアクセスし、サイトのアドレスを入力してください。
- **ドメインレジストラ**: URL（サイトアドレス）が登録されている会社です。ドメインレジストラを確認するには、[WhoIs.com](https://www.whois.com/whois/) にアクセスし、サイトのアドレスを入力してください。不正使用を報告するためのメールアドレスが表示される場合もあります。

## WooCommerce または WooCommerce.com はショップの名前です

これは次の 2 つのいずれかを意味します。

- WooCommerce.com で行われた正当な購入です。その場合、ご注文に含まれるすべての商品（ウェブサイト用のソフトウェア、プラグイン、テーマなど）は WooCommerce.com で見つかるはずです。WooCommerce.com アカウントにログインして、注文を管理できます。
- ストアオーナーが WooCommerce または WooCommerce.com という名称を無断で使用している可能性があります。その場合は、商標権侵害の可能性についてご連絡ください。

WooCommerce を利用したスト​​アで不快な思いをされたとのこと、大変残念に思います。ご満足いただける形で問題が解決することを願っております。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
