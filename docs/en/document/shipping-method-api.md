WooCommerce has a **shipping method API** which plugins can use to add their own rates. This article will take you through the steps to creating a new shipping method and interacting with the API.

Learn more about the Shipping Method API in the [WooCommerce Developer Documenation](https://developer.woocommerce.com/docs/shipping-method-api/).

- [Create a plugin](https://developer.woocommerce.com/docs/shipping-method-api/#0-create-a-plugin)
- [Create a function to house your class](https://developer.woocommerce.com/docs/shipping-method-api/#1-create-a-function-to-house-your-class)
- [Create your class](https://developer.woocommerce.com/docs/shipping-method-api/#2-create-your-class)
- [Defining settings/options](https://developer.woocommerce.com/docs/shipping-method-api/#3-defining-settings-options)
- [The calculate_shipping() method](https://developer.woocommerce.com/docs/shipping-method-api/#4-the-calculate-shipping-method)
- [Piecing it all together](https://developer.woocommerce.com/docs/shipping-method-api/#5-piecing-it-all-together)
