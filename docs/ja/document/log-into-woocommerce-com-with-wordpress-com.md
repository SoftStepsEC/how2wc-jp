WooCommerce.com にログインするには、WordPress.com アカウントが必要になります。どちらも [Automattic](https://automattic.com/about/) のサービスです。

つまり、次のことが可能になります。

- 1つのユーザー名とパスワードで、両方のサイトとアカウントに接続してアクセスできます。
- ログインとチェックアウトがさらに高速になります。
- 強化されたセキュリティをお楽しみください。

新しい WooCommerce.com アカウントを作成したいが、WordPress.com アカウントをお持ちでない場合は、次の手順に従ってください。

1. WooCommerce.com ホームページの右上にある「**ログイン**」をクリックします。
2. 次のページで、ページ右上にある「**アカウントを作成**」リンクをクリックします。
3. メールアドレスを入力し、ユーザー名とパスワードを作成して、WordPress.com アカウントに登録します。登録が完了したら、「**続行**」をクリックします。
4. 送信されたアクティベーションメールを開き、詳細を確認してアカウントをアクティベートします。

WooCommerce.comにログインしました。*マイアカウント*ダッシュボードから、[ダウンロード](https://woocommerce.com/my-account/downloads/)、[注文](https://woocommerce.com/my-account/orders/)、[サブスクリプション](https://woocommerce.com/my-account/my-subscriptions/)、[サポート](https://woocommerce.com/my-account/contact-support/)にアクセスできます。

新しい WooCommerce.com アカウントを作成し、既存の WordPress.com アカウントがある場合:

1. WooCommerce.com ホームページの右上にある「**ログイン**」をクリックします。
2. WordPress.com アカウントのユーザー名またはメールアドレスを入力し、「**続行**」をクリックします。
3. 次に、パスワードを入力し、「**ログイン**」をクリックします。
4. 「**続行**」をクリックして、現在の WordPress.com アカウントに接続します。別のアカウントに接続する場合は、「**別のアカウントでログイン**」をクリックします。
5. 最後に、「**承認**」をクリックして、WordPress.com アカウントを WooCommerce.com へのログインに承認します。

WooCommerce.comにログインしました。*マイアカウント*ダッシュボードから、[ダウンロード](https://woocommerce.com/my-account/downloads/)、[注文](https://woocommerce.com/my-account/orders/)、[サブスクリプション](https://woocommerce.com/my-account/my-subscriptions/)、[サポート](https://woocommerce.com/my-account/contact-support/)にアクセスできます。

**注:** WordPress.com アカウントのパスワードを紛失または忘れた場合は、「**パスワードを忘れましたか？**」をクリックし、メールアドレスを入力して「**パスワードをリセット**」をクリックしてください。すぐにログインするためのマジックリンクをメールでお送りします。WordPress.com の [パスワードを忘れた場合](https://wordpress.com/wp-login.php?action=lostpassword) ページにアクセスしてパスワードをリセットすることもできます。

**最近、WordPress.com アカウントを作成せずにゲストとして WooCommerce.com で購入された場合**、メールの受信トレイをご確認ください。WooCommerce から、メールアドレスの確認と WooCommerce.com アカウントの有効化を求めるメールが届いているはずです。

WooCommerce.com ダッシュボードには、アカウント有効化の通知も表示されます。こちらから有効化メールを再送信することもできます。

このメールは、WordPress.com アカウントを作成し、購入時に使用した WooCommerce.com アカウントに自動的にリンクするのに役立ちます。

WooCommerce.com アカウントをお持ちで、既存の WordPress.com アカウントをお持ちでない場合は、ログインできません。サブスクリプションを管理し、購入に関するサポートを受けるには、まず WordPress.com アカウントを作成する必要があります。

1. 受信したアカウント有効化メールで、「**メールアドレスを確認**」をクリックします。WooCommerce.com のログインページに移動します。
2. ページ右上の「**アカウントを作成**」リンクをクリックします。
3. メールアドレスを入力し、「**続行**」をクリックして WordPress.com アカウントにサインアップします。**重要:** WooCommerce.com での購入に使用した **メールアドレス** と同じものをご使用ください。
4. 次のページに、「*このメールアドレスは既にアカウントに関連付けられています*」というメッセージが表示されます。「**パスワードをお忘れですか?**」をクリックします。
5. 次に、メールアドレスを入力し、「**パスワードをリセット**」ボタンをクリックします。
6. メールの受信トレイを確認すると、WordPress.com からパスワードリセットメールが届いているはずです。「**パスワードをリセット**」ボタンをクリックします。
7. 次のページでパスワードを作成し、「**パスワードをリセット**」をクリックしてから、「**ログイン**」をクリックします。
8. WordPress.com アカウントのユーザー名とパスワードを使ってログインするか、メールでマジックリンクを受信して​​すぐにログインすることもできます。
9. ログインしたら、[WooCommerce.com](https://woocommerce.com/) に戻ります。

WooCommerce.comにログインしました。*マイアカウント*ダッシュボードから、[ダウンロード](https://woocommerce.com/my-account/downloads/)、[注文](https://woocommerce.com/my-account/orders/)、[サブスクリプション](https://woocommerce.com/my-account/my-subscriptions/)、[サポート](https://woocommerce.com/my-account/contact-support/)にアクセスできます。

上記の手順を実行してもアカウントのリンクに関してサポートが必要な場合は、[お問い合わせ](https://woocommerce.com/contact-us/)ください。

3. 次に、パスワードを入力し、「**ログイン**」をクリックします。
4. 「**続行**」をクリックして、現在の WordPress.com アカウントに接続します。別のアカウントに接続する場合は、「**別のアカウントでログイン**」をクリックします。
5. WooCommerce.com アカウントにリダイレクトされます。

WooCommerce.comにログインしました。*マイアカウント*ダッシュボードから、[ダウンロード](https://woocommerce.com/my-account/downloads/)、[注文](https://woocommerce.com/my-account/orders/)、[サブスクリプション](https://woocommerce.com/my-account/my-subscriptions/)、[サポート](https://woocommerce.com/my-account/contact-support/)にアクセスできます。

WooCommerce.com アカウントを切り替えるには:

1. WooCommerce.com アカウントのダッシュボードで、ページ右上にある **Gravatar プロフィール画像** をクリックします。
2. アカウント管理メニューが表示されるので、**ログアウト** をクリックします。
3. 次に、WooCommerce.com のホームページに移動し、右上にある **ログイン** をクリックします。
4. 次の画面で、**別のアカウントでログイン** をクリックし、別の WooCommerce.com アカウントのログイン情報を入力して続行します。

**ヒント:** 一時的に別のアカウントにログインしたい場合は、代わりに [プライベート ブラウザ ウィンドウ](https://www.wikihow.com/Activate-Incognito-Mode) (またはまったく別のブラウザ) を使用することもできます。

複数のクライアントアカウントを管理しているユーザーは、クライアントに[アカウント共同編集者として追加](https://woocommerce.com/document/adding-collaborators/)することを強くお勧めします。これにより、クライアントに代わってWooCommerce.comアカウントを管理する作業が簡単になります。

まだ質問があり、サポートが必要ですか?

- ヘルプデスクからハピネスエンジニアに[お問い合わせ](https://woocommerce.com/my-account/create-a-ticket/)ください。WooCommerce.comで開発または販売されている拡張機能、およびJetpack/WordPress.comをご利用のお客様には、サポートを提供しています。
- お客様でない場合は、[WooCommerceサポートフォーラム](https://wordpress.org/support/plugin/woocommerce/)でサポートを探すか、Wooエージェンシーパートナーにご依頼いただくことをお勧めします。これらのエージェンシーは、高度にカスタマイズされ、拡張性の高いオンラインストアの構築で実績のある信頼できるエージェンシーです。[Wooエージェンシーパートナーの詳細](https://woocommerce.com/development-services/)
