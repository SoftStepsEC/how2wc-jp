WooCommerce は、アクティブ化されると 2 つのユーザー ロールを登録し、既存の **Admin** ユーザー ロールに追加の権限を付与します。

- WooCommerce 設定用の `manage_woocommerce`
- `view_woocommerce_reports`

チェックアウトまたはサインアップのプロセスを通じて登録すると、ユーザーは **顧客** になります。

顧客:

- ほとんどの機能において**読み取りアクセス**のみ可能です。これは通常のブログ購読者と同等のステータスです。
- 自分の**アカウント** **情報**を編集できます。
- **過去/現在の注文**を閲覧できます。

**ショップマネージャー** は、管理者ではなくショップの管理を任せることができる役割です。ショップマネージャーには顧客が持つすべての権限が付与され、以下の主な権限が付与されます。

- `manage_woocommerce`: ショップマネージャーに、WooCommerce 内のすべての設定を管理し、商品を作成/編集するオプションを提供します。
- `view_woocommerce_reports`: すべての WooCommerce レポートへのアクセスを提供します。

**ショップマネージャーロール** には、[WooCommerce GitHub リポジトリ](https://github.com/woocommerce/woocommerce/blob/ee01d4219282387c2975ef4594677453c1dd7a0e/includes/class-wc-install.php#L1052) に記載されているすべての機能があります。

一般的な WordPress エディター機能も備えています。

役割と権限を変更したい場合は、プラグイン[ユーザー役割エディター - ユーザー役割の追加、割り当て、切り替え](https://woocommerce.com/products/user-roles-editor-for-woocommerce/)をお試しください。

次のようなサードパーティ製プラグインを使用することもできます。

- [Capability Manager Enhanced](http://wordpress.org/plugins/capability-manager-enhanced/)
- [メンバー](http://wordpress.org/plugins/members/)
- [グループ](http://wordpress.org/extend/plugins/groups/)

ただし、WooCommerce サポートはサードパーティ関連の問題やリクエストについてサポートを提供する立場に**ありません**のでご了承ください。この点については、[サポートポリシー](https://woocommerce.com/support-policy/#what-our-support-service-does-not-cover)にも記載されています。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
