ストアのアカウントをお持ちのお客様は、WooCommerceストアの顧客向けダッシュボード「**マイアカウント**」にアクセスできます。ここでは、過去の注文履歴を確認したり、住所や支払い情報を編集したりできます。このガイドでは、お客様がアカウントダッシュボードでできることの概要を説明します。

顧客がログインする前に「*マイ アカウント*」ページにアクセスすると、ログイン プロンプトが表示されます。

ログイン後、顧客にはデフォルトの*マイ アカウント* **ダッシュボード** とメニューが表示され、次の情報を表示するためのリンクが表示されます。

- 注文
- ダウンロード
- 住所（請求先と配送先）
- お支払い方法
- アカウント情報

このメニューからアカウントからログアウトすることもできます。

上の画像はWooCommerceのデフォルトのセクションを示していますが、プラグインやテーマを使用して、必要に応じてここにセクションを追加することもできます。例えば、[WooCommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/)を使用している場合、顧客には「サブスクリプション」という追加のメニュー項目が表示されます。

**注:** 図のように左側に *マイ アカウント* メニューを表示するには、次の **両方** が満たされている必要があります。1. *マイ アカウント* ページ エディターで、テンプレートを **デフォルト** (**全幅** ではない) に設定します。2. カスタマイザーの **レイアウト** で、右側に狭いサイドバーを表示するオプションを選択します。

「注文」セクションでは、お客様は注文履歴にアクセスできます。注文番号、日付、ステータス、合計金額、各注文で利用可能なアクションが一目で確認できます。

お客様は、リンクされた注文番号または注文の右側にある表示ボタンをクリックすることで、各注文の詳細画面にアクセスできます。注文のステータスによっては、追加の操作が利用できる場合があります。

*支払い失敗* または *支払い保留中* ステータスの注文の横には、**支払う** と **キャンセル** の 2 つの追加ボタンがあり、顧客は後でその注文に戻って支払いを行うことができます。

- [キャンセル] をクリックすると、注文は直ちにキャンセルされ、注文を完了したり、すぐに再注文したりする機会がなくなります。

- *支払う*ボタンをクリックすると、支払いの詳細のみを入力する簡略化されたチェックアウト ページが読み込まれ、注文を迅速に完了できます。

*注文詳細*ページで、お客様は**「再注文」**ボタンをクリックして、以前に**完了**した注文（つまり、*完了*ステータスになっている必要があります）を再注文できます。このページには、リンクされた注文番号または*表示*ボタンからアクセスできます。

*再度注文* ボタンをクリックすると、カート内の注文アイテムが再作成され、顧客はカート ページに移動します。そこで、顧客は編集したり、チェックアウトに進むことができます。

ダウンロード可能な製品の一部として購入された製品の場合、**ダウンロード** セクションには製品名、残りのダウンロード数、有効期限がリストされ、添付ファイルをダウンロードするためのボタンが表示されます。

顧客は、[住所] セクションから**請求先**住所と**発送先****住所を更新できます。

編集アイコンをクリックすると、請求または配送用の氏名、住所、会社名、電話番号、電子メール アドレスなど、チェックアウト ページに通常表示されるフィールドのセットが表示されます。

このセクションの詳細は請求と配送のみを目的としており、アカウントの詳細には影響しません。

ストアで**[サポートされている支払いゲートウェイ](https://woocommerce.com/document/advanced-payment-gateway-features/#section-9)が有効になっている場合、**支払い方法**で、顧客は保存されているクレジットカードの詳細を見つけることができます。

**注:** このページに表示される支払い方法の数は、現在、WP Admin の *設定 > 表示* の **ブログページを最も多く表示する** 設定によって異なります。

**アカウント詳細**セクションでは、お客様はユーザーアカウントに関連する個人情報を更新できます。詳細情報には、氏名（姓名）、表示名（**アカウントセクション**と**レビュー**で使用されます）が含まれます。

顧客はこのセクションでアカウントのパスワードをリセットすることもできます。

このリンクをクリックすると、顧客はアカウントからログアウトされ、ダッシュボード セクションに戻り、ログイン プロンプトが再度表示されます。
