ここでは、Woo モバイル アプリで製品画像やダウンロード可能なファイルをアップロードしようとしたときに発生する可能性のあるエラー メッセージをいくつか紹介します。

これは通常、アップロードしようとしているファイルがサーバーの制限を超えていることを意味します。この問題を確認する最も簡単なテストは、画像のサイズを小さく（1MB未満）し、.jpgや.pngなどのWeb対応形式で保存してから、再度アップロードしてみることです。

この問題を解決するには**2つの方法**があります:

特定の画像ファイルは、サイズが大きすぎるためにアップロードできない場合があります。この問題を解決するには、画像エディタまたはオンラインツールを使用して画像のサイズを調整し、小さくしてください。詳しくは、[WooCommerce に最適な画像サイズ](https://woocommerce.com/document/adding-product-images-and-galleries/#what-is-the-ideal-image-size-to-use-with-woocommerce) をご覧ください。

サイトの最大 (max) ファイル サイズを増やすには、2 つの方法があります。

- ホスティング会社に直接連絡し、ファイルサイズの制限を増やすよう依頼してください。
- サイトの技術的な更新に問題がない場合は、FTP 経由で WordPress のメインフォルダにアクセスし、`php.ini` ファイルを探してください。ファイルを開き、**以下のテキストを追加** してから保存してください。(`php.ini` が見つからない場合は、新規に作成し、必要なテキストを貼り付けて、WordPress のメインフォルダにアップロードしてください。)

```
upload_max_filesize = 128M
post_max_size = 128M
max_execution_time = 300
```

これにより、サイトの最大アップロード制限が増加し、画像をアップロードできるようになります。

**注**: 一部の****[WordPressホスト](https://woocommerce.com/hosting-solutions/)****では、アップロードサイズの上限を手動で増やすことが制限されている場合があります。手動で増やすことができなかった場合は、ホストに連絡して制限の変更をリクエストする必要があります。

MIME（MultipurposeInternetMailExtensions）タイプは、ブラウザやその他のインターネットデバイスで、ページに関連付けられたコンテンツの種類を判断するために使用されます。例えば、ページに.pngファイルと.jpegファイルがある場合、ブラウザはMIMEタイプに基づいて、両方のファイルを動画などのファイル形式ではなく画像として扱います。

WooCommerce では、WordPress で定義されているものと同じファイル拡張子を使用できます。完全なリストは、[WordPress Codex: ファイルのアップロード](https://codex.wordpress.org/Uploading_Files) で確認できます。アプリ経由で特定のファイルをアップロードできない場合は、MIME タイプがサポートされていない可能性があります。許可されていないファイルタイプの問題を回避するには、ファイルを作成した元のアプリで開き、.jpg や .png などの Web 対応形式にエクスポートしてみてください。Web からダウンロードした場合は、.webp 画像はサポートされていないため、ダウンロードする前に画像をタップして新しいブラウザウィンドウで表示してみてください。それでも問題が解決しない場合は、サポートチーム（mobile-support@woocommerce.com）に連絡して問題を報告してください。その際、デバイスの詳細、画像の作成方法、アップロード前に画像が編集されたかどうか、そして可能であればアップロードに失敗したサンプルファイルのコピーを添付してください。

最大ファイルアップロード制限を超えるファイルをアップロードすると、アップロードが失敗し、エラーが発生します。最大ファイルアップロード制限は2MBから150MBまでで、サイトのホスティングパッケージによって異なります。

**アップロードファイルの最大サイズを確認する方法**

アップロードファイルの最大サイズを確認するには、WordPressダッシュボードから「メディア」>「新しいメディアファイルを追加」をクリックします。スクリーンショットに示すように、アップロードウィンドウの下部に最大アップロードファイルサイズが表示されます。

アップロードファイルの最大サイズを増やすには、「最大ファイルサイズを増やす方法」のセクションを確認してください。

アップロードしようとしているファイルサイズが、サイトに設定されている最大メモリ制限を超えている可能性があります。サイズの小さい画像をアップロードするか、ストアのWordPressメモリ制限を増やすことをお試しください。WordPressメモリ制限を増やすには、[こちらのドキュメント](https://woocommerce.com/document/increasing-the-wordpress-memory-limit/)をご覧ください。

ストレージ容量とは、ウェブサイトにさまざまなファイルや画像をアップロードする際に利用できるディスク容量のことです。デスクトップパソコンのディスク容量に似ていますが、パソコン内ではなくWordPress.comのサーバー上に保存されます。

WordPress.com でサイトをホスティングしている場合は、「**マイサイト**」（複数のサイトがある場合は「マイサイト」）に移動し、左側のサイドバーから「**メディア**」を選択してください。ストレージの上限と使用済みの容量が表示されます。

WordPress.com の標準無料アカウントには3GBのストレージ容量が付属しています。追加のストレージ容量が必要な場合は、[アカウントを有料プランにアップグレード](https://wordpress.com/pricing/)してください。画像や動画を削除して容量を増やすこともできますが、ご自身のサイトまたは他のサイトにリンクされているメディアファイルを削除すると、以前それらのファイルを参照していたコンテンツのリンクが切れてしまうことにご注意ください。

これは、アプリがアップロードが必要なファイルのパスを見つけられないことを意味します。画像をデバイス上の新しい場所に移動または再保存してから、もう一度お試しください。

アプリがアクセスしようとしているファイルはデバイス上で見つかりません。画像をデバイス上の新しい場所に移動または再保存して、もう一度お試しください。

アプリにはファイルを読み取るための十分な権限がありません。

上記の3つの問題は、デバイスから画像を選択する際に外部ファイルマネージャーを使用することで発生します。上記のいずれかの問題が発生した場合は、アプリ内から「マイストア」>「設定アイコン」>「ヘルプとサポート」にアクセスし、お問い合わせフォームにご記入の上、サポートチームまでご連絡ください。画像をデバイス上の別の場所に移動または再保存して、もう一度お試しください。問題が解決する場合があります。

デバイスのインターネット接続が不安定な場合、タイムアウトや接続の問題が発生する可能性があります。問題の原因を特定するための手順をいくつかご紹介します。

- 別の接続で画像のアップロードをお試しください。Wi-Fi を推奨します。
- インターネット接続が安定している場合は、モバイル端末のブラウザから `https://YOURSITE.com/wp-admin/upload.php` (`YOURSITE` をサイトの URL に変更してください) にアクセスし、画像のアップロードをお試しください。Web からは画像のアップロードができてアプリからはできない場合は、サポートチームまでお問い合わせください。
- サイトのパフォーマンスに問題が発生している可能性があります。セキュリティプラグインがインストールされているかどうかを確認し、ホスティングプロバイダーに連絡して、サーバーログで当社のサーバーからのブロックされたリクエストを確認することをお勧めします。[このドキュメントで詳細を説明しています](https://woocommerce.com/document/android-ios-apps-troubleshooting-error-fetching-orders/#section-3)。

**注:** WooCommerce モバイル アプリは、ショップ所有者、管理者、ショップ マネージャーのアクセスのみをサポートします。

Jetpackプラグインを使用してモバイルアプリをサイトに接続している場合は、Jetpack接続がアクティブで正常に機能していることを確認してください。[よくある問題を確認する](https://jetpack.com/support/getting-started-with-jetpack/known-issues/)、または[サイトを再接続する](https://jetpack.com/support/reconnecting-reinstalling-jetpack/)を行ってください。

それでも問題が解決しない場合は、*メニュー > 設定 > ヘルプとサポート > サポートに問い合わせ* して、アプリ内からサポートにお問い合わせください。
