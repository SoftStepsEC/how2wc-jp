適切なWooCommerceテーマの選択は、当然ながら主観的な要素が強くなります。「美しさは見る人の目の中にある」という諺がありますが、美しさは表面的なものです。ストアのテーマを選ぶ前に考慮すべきヒントをいくつかご紹介します。

[テーマの全リスト](https://woocommerce.com/product-category/themes?categoryIds=1033&collections=theme&page=2&orderby=date)をご覧ください↗

「見た目が良い」というテーマだけを探すのはやめましょう。適切なテーマを選ぶ必要があります。フェラーリは見た目は素晴らしいですが、買い物に出かけたり帰ったりするのに最適な車ではありません。

ターゲットオーディエンス、ブランド、そして何よりも商品について考えてみましょう。これら3つの要素に何らかの形で共鳴するテーマを選ぶようにしましょう。[ミニマリスト家具](https://woocommerce.com/product-category/themes?categoryIds=3777&collections=theme&page=1&orderby=date)を販売しているなら、余白とすっきりとしたタイポグラフィのあるテーマを探しましょう。[子供のおもちゃ](https://woocommerce.com/product-category/themes?categoryIds=3780&collections=theme&page=1&orderby=date)を販売しているなら、明るい色をふんだんに使った遊び心のあるテーマを探しましょう…といった具合です。

検討中のテーマがWooCommerce対応を謳っている場合は、WooCommerceページをすべて確認し、確実に機能していることを確認してください。「マイアカウント」セクション、カート、チェックアウトなどのページも忘れずにチェックしましょう。

WooCommerceとWordPressはどちらも、年に数回の「大規模」リリースを含む、ある程度定期的にアップデートをリリースしています。これらのアップデートは、最新のベータ版で十分にテストされていない場合、テーマにエラーを引き起こす可能性があります。テーマの変更履歴を確認し、開発者が定期的にメンテナンスを行っているかどうかを確認することをお勧めします。また、テーマをインストールする前に、そのテーマがWordPressとWooCommerceの最新バージョンと互換性があることを確認してください。

「WooCommerce対応」テーマが最新バージョンのWooCommerceと本当に互換性があるかどうかを確認するには、以下の簡単なチェックを実行してください。デモ版をご覧になり、ソースコードを確認してください。ソースコードには、アクティブなWooCommerceのバージョンを宣言する「generator」メタタグがあります。以下のようになっているはずです。

```
<!-- WooCommerce Version -->
<meta name="generator" content="WooCommerce 2.0.13" />
```

デモ版がWooCommerceの最新バージョンを実行していることを確認してください。数バージョン古い場合は理由を尋ねてください。バージョンがかなり古い場合は注意が必要です。

ほぼすべてのWordPressテーマと大規模プラグインには、時折バグが存在します。バグを見つけた場合は、すぐに修正されるか、あるいは少なくとも自分で作り出したバグであれば修正方法を案内されるかを確認する必要があります。

プレミアムテーマを購入する場合は、テーマショップのサポートポリシーを確認してください。WordPress.orgのテーマを使用している場合は、サポートフォーラムの活動状況を確認してください。

テーマにどの程度のカスタム機能が組み込まれているかも確認しておきましょう。テーマ作成者の中には（ThemeForestの作者はこの点で有名です）、膨大な数の機能をテーマに組み込んでいる人もいます。

カスタムスライダー、ショートコード、ウィジェット、レイアウトマネージャー、コンテンツビルダーといった機能が頻繁に含まれています。このようなテーマを選ぶ際には注意が必要です。後日テーマを変更すると、これまで使い慣れていたカスタム機能がすべて失われてしまう可能性があります。特に、ウェブサイト全体にカスタム機能が組み込まれている場合は、非常に大きな問題となる可能性があります。例えば、すべての商品ページで [slider] ショートコードを使用しているとします。テーマを変更すると、すべての商品ページからそのショートコードを削除しなければならなくなり、非常に手間がかかります。これは決して楽なことではありません！

常に軽量で無駄のないテーマを探し、プラグインや拡張機能を使って機能を追加することをお勧めします。そうすれば、テーマを切り替えても機能はそのまま維持されます。

テーマを選ぶ際に最後に考慮すべき点は、「レスポンシブ」かどうかです。レスポンシブデザインとは、スマートフォンからデスクトップパソコンやテレビまで、あらゆるデバイスに最適化されていることを意味します。

これもやはりオーディエンス次第ですが、Webは急速に多様化しており、人々は様々なデバイス（スマートフォン、テレビ、電子レンジなど）を使ってWebを閲覧していることは留意しておくべきです。すべての顧客に素晴らしい体験を提供したいはずです。

[レスポンシブテーマの完全なリスト](https://woocommerce.com/product-category/themes?categoryIds=1051&collections=theme&page=1&orderby=date)をご覧ください↗

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
