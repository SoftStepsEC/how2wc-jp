テンプレートとは、複数の投稿、ページ、商品の外観とレイアウトを決定するブロックの配置です。**カスタムテンプレート**を作成することも、テーマやプラグインに付属の**テンプレート**を使用することもできます。テンプレートに変更を加えると、エディターはそのテンプレートを使用しているすべてのページ/投稿のブロックを更新します。

[テンプレートエディターの詳細](https://wordpress.org/support/article/template-editor/)をご覧ください。

WooCommerce には、ストアの外観を制御するさまざまなテンプレートが含まれています。

利用可能なすべての WooCommerce テンプレートを表示するには:

1. WordPress管理ダッシュボードから、**「外観」>「エディター」**に移動します。
2. **「テンプレート」**をクリックします。
3. 左側のサイドバーメニューの下部にある**「すべてのテンプレートを管理」**をクリックします。
4. 左側のサイドバーで、**「WooCommerce」**をクリックします。

あるいは、[エディター コマンド パレット](https://wordpress.com/support/site-editor/command-palette/) から任意のテンプレートに直接アクセスすることもできます。

1. WP Admin ダッシュボードから、**「外観」>「エディター」** に移動します。
2. キーボードで、Cmd+K (MacOS) または Ctrl+K (Windows) を押します。
3. すでにテンプレートまたはテンプレートパーツを編集している場合は、**テンプレートのタイトルをクリック** してコマンドを検索することもできます。

WooCommerce には、すぐに使えるカスタマイズ可能なテンプレートのコレクションが含まれています。

- 単一商品
- 商品カタログ
- 商品検索結果
- 属性別商品
- カテゴリー別商品
- タグ別商品
- 注文確認
- ページ: カート
- ページ: チェックアウト

これらのテンプレートは**カスタマイズ可能**です。ブロックを移動したり、周囲にブロックを追加したりすることで、テンプレートを拡張・カスタマイズできます。また、特定の商品や特定の商品カテゴリー向けにカスタムテンプレートを作成することもできます。

テーマに上記のテンプレートのいずれかが含まれている場合、それらは WooCommerce のデフォルトのテンプレートを **自動的に上書き** します。

**開発者の皆様へ:** カスタムブロックテーマでデフォルトのWooCommerceブロックテンプレートを上書きしたい場合は、これらのテンプレートのカスタマイズバージョンをテーマの `/templates` ディレクトリに配置してください。[GitHubでWooCommerceブロックテンプレートファイルを確認してください](https://github.com/woocommerce/woocommerce/tree/trunk/plugins/woocommerce/templates/templates/blockified)。[テンプレートの詳細](https://developer.wordpress.org/themes/templates/templates/#how-do-templates-work)。

場合によっては、テンプレートのカスタマイズを進める最も簡単な方法は、以前の変更をすべて消去し、テンプレートをデフォルトの状態に戻すことです。これを行うにはいくつかの方法があります。

テンプレートのカスタマイズをクリアするには:

1. WP Admin ダッシュボードから、**「外観」>「エディター」** に移動します。
2. **「テンプレート」** をクリックします。
3. 左側のサイドバーメニューの下部にある **「すべてのテンプレートを管理」** をクリックします。
4. カスタマイズされたテンプレートには、**時計** の周りに反時計回りの矢印が付いたアイコンが表示され、「**リセット**」というラベルが付けられます (下の画像を参照)。
5. このアイコンをクリックすると、そのテンプレートのカスタマイズがクリアされます。

あるいは、次のこともできます。

1. **「外観 > エディター」** からエディターを開きます。
2. **「テンプレート」** をクリックします。
3. クリアしたいカスタマイズを含むテンプレートを選択して開きます。
4. テンプレート名の横にある3つの点の **ケバブメニューアイコン** (⁝) をクリックします。
5. **「カスタマイズをクリア」** をクリックします。

テンプレートに対して行ったカスタマイズをクリアすると、テンプレートは**デフォルトの状態**に戻ります。
