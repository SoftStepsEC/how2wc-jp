WooCommerceのComing Soonページテンプレートは、[サイトエディタ](https://woocommerce.com/document/woocommerce-store-editing/the-editor/)で完全にカスタマイズ可能で、ブランドイメージに合わせてデザインを変更できます。色、フォント、レイアウトを調整することで、魅力的な外観を維持しながら、ストアのアイデンティティを反映したComing Soonページを作成できます。

### WooCommerce 9.8 以降の新しい近日公開デザイン

WooCommerce 9.8では、新しい「近日公開」デザインが導入されました。販売者は、美しくデザインされた以下のデザインからお選びいただけます。

1. **デフォルトの近日公開** – ロゴとメッセージが表示される、紫色のフルスクリーンデザイン。
2. **画像ギャラリー** – 複数の画像で商品のプレビューを表示できる、多用途のギャラリースタイルのデザイン。ストアオープン時に新商品のプレビューを訪問者に提供したいマーチャントに最適です。ギャラリーレイアウトでは、さまざまな商品や機能を同時に強調表示できるため、カタログ全体への期待を高めることができます。
3. **ヘッダーとフッター付き近日公開** – サイトのヘッダーとフッターを維持しながら、近日公開コンテンツを表示します。
4. **最小限の左画像** – すっきりとした2列構成の近日公開ページで、左側に大きな丸みを帯びた画像、右側に簡潔なリリースメッセージを配置します。広々としたレイアウトはシンプルさとエレガンスを強調し、洗練されたモダンなリリース前のプレゼンスを求めるブランドに最適です。
5. **モダンブラック** – 力強いビジュアルインパクトを生み出す、全幅の背景画像と中央に配置されたメッセージオーバーレイを備えた、大胆な近日公開ページです。ミニマルでありながら力強いデザインは、ドラマチックな第一印象を与えたいブランドに最適です。
6. **右分割画像** – コンテンツとビジュアルのバランスが取れた、ダイナミックな2カラムレイアウト。左側には小さな画像でまとめたコンパクトな告知を、右側には全高の背景画像とオーバーレイテキストを配置した大胆なレイアウトです。ストーリーテリングと力強いイメージを融合させたブランドに最適です。

このセクションでは、サイトエディターで「Coming Soon」テンプレートをカスタマイズする方法について説明します。これを行うには、サイトでブロックテーマを有効にし、以下の手順に従う必要があります。

1. *WooCommerce > 設定 > サイトの公開設定* に移動します。
2. **近日公開** ラジオボタンの下にある **エディター** リンクをクリックします。

**注記：**

**ストアページにのみ適用** トグルを有効にすると、近日公開ページではテーマのヘッダーとフッターを含む別のデザインが使用されます。

**注記：**

これらのデザインは WooCommerce 9.8 以降で利用できます。

1. サイトエディターで、右側のサイドバーにある**テンプレート**タブをクリックします。
2. **デザイン**をクリックします。
3. 利用可能なすべてのデザインがプレビューサムネイルとともに表示されます。任意のデザインをクリックして選択します。
4. 必要に応じて追加のカスタマイズを行い、完了したら**保存**をクリックします。

デフォルトの「近日公開」デザインでは、背景色を設定できるカスタムブロックが使用されています。背景色を変更するには、以下の手順に従ってください。

1. **ドキュメント概要** を開き、**近日公開** ブロックを選択します。
2. **設定** パネルを開き、**ブロック** タブを選択します。
3. 必要に応じてテキストと背景色を変更します。

画像ギャラリーのデザインには、美しいストック写真がプレースホルダーとして含まれています。これらのプレースホルダーは、ご自身の画像に置き換えることができます。

1. デザイン内の任意の画像をクリックします。
2. **置換**をクリックします。
3. **メディアライブラリ**または**アップロード**を選択して画像を選択します。

**Modern Black** デザインでは、Gutenberg の **Cover** ブロックを使用して、ページ全体に画像を配置しています。画像を置き換えるには、以下の手順に従ってください。

1. **ドキュメント概要** を開き、**カバー** ブロックを選択します。
2. **置換** をクリックします。
3. **メディアライブラリ** または **アップロード** を選択して画像を選択します。

サイトエディターを使用してテンプレートをカスタマイズする方法の詳細については、[WordPress 公式ドキュメント](https://wordpress.org/documentation/article/styles-overview/#how-to-find-styles)をご覧ください。
