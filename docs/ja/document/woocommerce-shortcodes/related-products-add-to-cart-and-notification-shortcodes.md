このページには、**関連商品セクション**、**価格表示付きのカートに追加ボタン**、WooCommerce以外のページに**ショップメッセージ/通知**を表示するためのショートコードが掲載されています。以下に各ショートコードとその属性を記載します。

クロスセルやアップセルを促進するために、**関連商品**（類似のタグやカテゴリーを持つ商品）を単一の商品ページに表示したい場合は、**[related_products] ショートコード** を使用できます。以下は、ショートコードと、それをフォーマットするために使用できる `attributes` です。

このショートコードをフォーマットするために使用できる属性は次のとおりです。

- **Limit :**このショートコードを使用して、ユーザーに表示される関連商品の数を制限します。例えば、[related_products limit =’12”] と指定すると、表示される商品の数が12個に制限されます。

- **列数:**関連商品を表示するために使用する列の数を決定します。例えば、[related_products columns=”4″] と指定すると、商品は4つの列（1行につき1商品）に配置されます。つまり、12という制限と合わせると、4商品ずつ3行表示されることになります。

- **Orderby:**関連商品セクションの商品の並び順を決定します。価格またはタイトルで並び替えることができます。

いくつかの `attributes` を設定してこのショートコードを使用し、そのショートコードを 1 つの商品の「商品の説明」セクションに配置する例を次に示します。

`[関連商品制限="4" 列="3" 順序="価格"]`

ショートコードを使えば、どのページにも「カートに追加」ボタンを表示できます。これは、ショップページ以外の場所で商品を目立たせたい場合に便利です。以下に、ショートコードと利用可能な属性、そして投稿での使用例を示します。

利用可能な属性は次のとおりです。

- **id :** これは商品IDです。**「商品 > すべての商品」** で商品にマウスオーバーすると、商品IDを確認できます。商品IDを「カートに追加」ショートコードに追加すると、その商品の価格と「カートに追加」ボタンが表示されます。
- **style**: これは「カートに追加」ボタンの一般的なスタイルです。枠線、枠線の種類/色、パディングを設定できます。
- **sku**: これは商品のSKUです。商品IDを設定しない場合、ショートコードはSKUを使用して適切な「カートに追加」ボタンを表示します。
- **show_price :**「カートに追加」ボタンの横に価格を表示するには、「TRUE」または「FALSE」に設定します。
- **class :**追加のCSSを追加します。
- **quantity**: カートに追加された数量を決定します。(**注:** この属性は、テーマテンプレート経由でショートコードを追加した場合にのみ機能します。)

```
[add_to_cart id="99"]
```

コードの例を以下に示します: `[add_to_cart id="xx" style="border:none; padding 10px"]` が投稿に追加されました (**注意:**[add_to_cart] はボタンと商品の価格のみを追加し、画像や追加の商品情報は追加しません)。:

**注:** 「*xx*」を、カートに追加ボタンを表示する製品の製品 ID に置き換えます。

ID 別に、単一商品の「カートに追加」ボタンに URL を表示します。

このショートコードには、**id** または **sku** の2つの属性が使用できます。前の例の [add_to_cart] ショートコードでも同じ属性が使用されています。

```
[add_to_cart_url id="99"]
```
