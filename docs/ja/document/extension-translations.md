---
title: 拡張翻訳

---

WooCommerce が提供する拡張機能の複数言語への翻訳を開始しており、まずはスペイン語から開始します。対応言語は各商品ページでハイライト表示され、国旗が表示されるので、その地域で拡張機能がサポートされているかどうかが一目でわかります。

ベンダーに拡張機能の翻訳を強制することはありません。拡張機能の翻訳を提供したいベンダーは、ホスト言語で拡張機能を提出することで提供できます。

WooCommerce は拡張機能をスペイン語、フランス語、ポルトガル語に翻訳する予定です。現時点では、これらの言語以外への拡張は予定しておりません。

まずは [WooCommerce Bookings](https://woocommerce.com/products/woocommerce-bookings/) から始めましょう。これには、付随する拡張機能 [WooCommerce Bookings Availability](https://woocommerce.com/products/bookings-availability/) と [WooCommerce Accommodation Bookings.](https://woocommerce.com/products/woocommerce-accommodation-bookings/) も含まれます。

サードパーティ製の拡張機能は翻訳を必須としません。開発者は複数の言語で拡張機能を提出できますが、その言語でのサポートが保証されるものではありません。各ベンダーのサポートガイドラインをご確認ください。

WooCommerceはすぐにローカライズ可能です。[詳細な翻訳手順](https://woocommerce.com/document/woocommerce-localization/#section-2)をご確認ください。

WooCommerce と WordPress はユーザーの皆様の貢献によって成長しています。必要なのは、あなたの言語への翻訳ファイルだけです。[翻訳を送信](https://translate.wordpress.org/)

翻訳が初めての方は、[WordPress 翻訳ハンドブック](https://make.wordpress.org/polyglots/handbook/tools/glotpress-translate-wordpress-org/) を確認して始めましょう。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
