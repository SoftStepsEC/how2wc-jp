WooCommerce モバイルアプリの **注文** タブにアクセスするには、ナビゲーションバーの左下にある **注文アイコン** をタップしてください。**注文** タブでは注文リストにアクセスでき、ステータスで絞り込んだり、特定の注文を検索したりできます。注文をクリックすると、表示または管理できます。

注文を作成するには、[このページのドキュメント](https://woocommerce.com/document/creating-orders-on-the-mobile-app/)をお読みください。

#### 順序フィルター

注文は**注文ステータス**や**日付範囲**でフィルタリングできます。

1. 画面右上の「フィルター」をタップします。
2. 「注文ステータス」を選択して、以下の条件で絞り込みます：すべて、保留中、処理中、保留中、失敗、キャンセル済み、完了、返金済み
3. 「日付範囲」を選択して、以下の条件で絞り込みます：すべて、今日、過去2日間、過去7日間、過去30日間、カスタム範囲（開始日と終了日を選択）
4. 「顧客」を選択して、特定の顧客の注文を表示します。
5. 「商品」を選択して、特定の商品のみを含む注文を表示します。
6. 各フィルターを選択した後、「注文を表示」をタップしてフィルターを適用します。
7. フィルターが有効な間、「注文」ページにフィルター数が表示され、「フィルター表示」に「すべてクリア」オプションが表示されます。「すべてクリア」をタップしてから「注文を表示」をタップすると、すべての注文フィルターがリセットされます。

特定の注文を**検索**するには:

1. 上部のツールバーにある**検索**アイコンをタップします。
2. 入力を開始します。顧客の名、姓、製品、注文番号で一致する用語が検索されます。

- 検索アイコンを押すと検索オプションが表示されます。
- 検索したい内容を入力し、「完了」を押すと、一致する結果が表示されます。

#### 注文の表示と管理

**注文を表示**

注文の詳細を表示するには、リストで個々の注文を選択するか、「新規注文」通知に表示されている注文をタップします。

**注文詳細** 画面には次の詳細が表示されます。

- 注文日、注文番号、顧客名、注文ステータス
- 注文商品、数量、属性、価格のリスト
- お支払い情報
- 注文ステータスに関連するボタン（例：**処理中**の場合は「注文を完了としてマーク**」、注文が**処理中**または**完了**の場合は「返金**」）
- 顧客メモ
- 配送詳細
- 請求情報（顧客名、住所、メールアドレスなど）
- 注文メモ（非公開、システム、公開）
- 関連サブスクリプション（[WooCommerce Subscriptions](https://woocommerce.com/products/woocommerce-subscriptions/)拡張機能で作成されたサブスクリプションにのみ適用されます）

バージョン9.7以降、注文にカスタムフィールドが含まれている場合、アプリの注文詳細画面で注文商品の下に「**カスタムフィールドを表示**」ボタンが表示されます。ボタンをタップすると、カスタムフィールドのリストが表示されます。

- 右上のXを押すと、「注文の表示」画面に戻ります。

**注文を履行する**

1. **処理中** ステータスの注文を開き、**注文詳細** 画面を表示します。
2. **注文を完了する** をタップします。**注文内容を確認する** 画面が表示されます。
3. 商品の詳細と配送情報を確認し、**注文を完了する** をタップします。

アプリで注文を完了すると、サイトで注文が完了した場合と同じように、顧客に「**注文完了**」メールが送信されます。詳しくは[メール通知](https://woocommerce.com/document/configuring-woocommerce-settings/#section-19)をご覧ください。

**注:** 間違いがありましたか？注文を発送手続きへ送信すると、取り消しオプションを含むメッセージが*短時間*表示されます。「取り消し」をクリックすると、注文は元の状態に戻ります。

**製品の詳細を見る**

製品をタップして、注文内の製品の詳細ビューを開きます。

**注文メモを追加**

1. 注文を開き、「**注文メモ**」セクションまでスクロールします。
2. 注文メモを追加するには、「**+メモを追加**」を押します。
3. メモはデフォルトでは非公開ですが、「**顧客にメモをメールで送信**」設定を切り替えることで、顧客に公開できます。
4. 右上の「**追加**」をタップしてメモを保存します。

**顧客に連絡する**

1. 注文を開き、下にスクロールして「**顧客**」セクションに移動し、「**請求情報を表示**」を選択します。
2. 請求先住所を入力した場合は、住所エリアを長押しして「**コピー**」を選択するとコピーできます。
3. チェックアウト時に有効な電話番号を追加した場合は、右側の**省略記号メニュー**に表示されます。メニューをタップすると、顧客に**電話をかける**か**メッセージを送信する**オプションが表示されます（メッセージはすべての番号に送信されますが、携帯電話番号にのみ送信されます）。
4. チェックアウト時に有効なメールアドレスを追加した場合は、右側に**封筒アイコン**が表示されます。アイコンをタップすると、顧客にメールを送信できます。

**注文の払い戻し**

1. 返金（または一部返金）したい注文を開き、下にスクロールして「**返金**」ボタンを選択します。
2. 返金する各商品の数量を入力します。注文内のすべての商品を選択するには、右上の「**すべて選択**」を押します。
3. 送料を返金するには、「**送料返金**」のトグルを右にスライドします。
4. 「**次へ**」を押し、返金に関する注文メモを任意で残すことができます。
5. 「**返金**」を押すと、確認画面が表示されます。
6. 「**返金**」を押して確定するか、「**キャンセル**」を押して戻ります。「**返金**」を押して返金が処理されると、通知が表示されます。

iPadの「注文」タブは、より多くの詳細を一度に表示できるように最適化されており、注文リストと注文詳細が並べて表示されます。注文の作成と編集では、注文と同時にすべての商品を確認できます。

Jetpackプラグインを使用してアプリをサイトに接続している場合は、Jetpack接続がアクティブで正常に動作していることを確認してください。[既知の問題](https://jetpack.com/support/getting-started-with-jetpack/known-issues/)を確認するか、[サイトの再接続](https://jetpack.com/support/reconnecting-reinstalling-jetpack/)をお試しください。

それでも問題が解決しない場合は、アプリ内から「メニュー」>「設定」>「ヘルプとサポート」>「サポートに問い合わせ」にアクセスし、お問い合わせフォームにご記入の上、サポート チームにお問い合わせください。
