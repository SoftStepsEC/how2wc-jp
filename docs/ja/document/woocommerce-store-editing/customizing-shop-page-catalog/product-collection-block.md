製品コレクション ブロックは、製品のアーカイブを表示およびカスタマイズし、製品の推奨事項を提供できるように設計されています。

ブロックを挿入すると、現在編集中のテンプレートまたはページに応じて、独自のカスタム製品コレクションを作成するか、最も一般的なタイプの推奨事項をすばやく提供できるバリエーション/レシピのリストから選択するように求められます。

製品コレクション ブロックは、すべての WooCommerce テンプレートで製品を表示するためにデフォルトで使用されます。

- 製品カタログ
- カテゴリー別製品
- タグ別製品
- 属性別製品
- ブランド別製品
- 製品検索結果

製品コレクション ブロックは、それが含まれているテンプレート以外にも、サイト上の製品を紹介したい任意のページまたはテンプレートに追加できます。

商品コレクションブロックには、あらかじめ定義されたレシピが用意されており、そこから選択してさらにカスタマイズできます。あらかじめ用意されたコレクションレシピはバナーに最適で、デフォルトではページ区切りのない1行の商品のみが表示されますが、デザインやレイアウトに合わせて編集できます。カスタムコレクションを作成することもできます。

デフォルトでは、商品コレクションにはショップのカタログ全体が表示され、表示されているページのコンテキストに合わせて自動的に調整されます。例えば、「タグ別商品」テンプレートに商品コレクションを追加し、「独自に作成」を選択した場合、特定のタグでページにアクセスした買い物客には、そのタグが付いたカタログ内の商品のみが表示されます。商品コレクションがページのコンテキストに合わせて動的に調整されないようにするには、コレクションのサイドバーブロック設定で「**現在のクエリと同期**」設定をオンにします。

このコレクションを使って、おすすめ商品をアピールしましょう。[商品をおすすめ商品としてマークする](https://woocommerce.com/document/managing-products/feature-filter-and-sort-products/#mark-a-product-as-featured)の方法をご覧ください。

この製品コレクションを使用して、ショップ内の[最高のレビュー](https://woocommerce.com/document/product-reviews/)を持つ製品を推奨します。

名前の通り、この商品コレクションでは、ショップで販売されている商品を紹介します。

ベストセラー商品コレクションでは、サイト上で最も多く購入された商品を紹介します。同じ販売回数の商品がある場合、最も最近公開された商品が最初に表示されます。

新着商品コレクションでは、カタログに最近追加された商品がハイライト表示されます。デフォルトでは、過去7日以内に作成された商品のみが表示されます。この設定はブロックサイドバーで変更できます。

厳選商品コレクションを使えば、ご自身で厳選した商品をまとめて紹介できます。限定商品や、ご自身で特集したい商品セットを目立たせるのに最適です。

関連商品コレクションを使用すると、買い物客が既に閲覧している商品と関連する商品や類似商品を自動的におすすめできます。WooCommerce は、カテゴリやタグに基づいて [関連商品](https://woocommerce.com/document/related-products-up-sells-and-cross-sells/#related-products) を自動的に決定します。

アップセル製品コレクションを使用して、顧客が興味を持っている製品の上位バージョンまたはプレミアムバージョンを宣伝します。[製品編集](https://woocommerce.com/document/related-products-up-sells-and-cross-sells/)画面で[リンクされた製品](https://woocommerce.com/document/related-products-up-sells-and-cross-sells/)を追加します。

クロスセル商品コレクションを使用すると、顧客のカート内の商品と相性の良い商品をハイライト表示できます。クロスセル商品はカートページやチェックアウト画面に表示されることが多く、メインの購入商品と相性の良い商品を[おすすめ](https://woocommerce.com/document/related-products-up-sells-and-cross-sells/#cross-sells)します。

コレクションとは、基本的にコレクションブロックにプリセットされたフィルターです。ブロックツールバーの「コレクションを選択」ボタンをクリックすると、いつでも選択したコレクションを変更できます。その後、[ブロック設定サイドバー](https://woocommerce.com/document/woocommerce-store-editing/blocks/#block-settings)でフィルターを必要に応じて調整できます。

使用する製品コレクションを選択すると、リストビューに、選択したコレクションに一致する親ブロックが追加され、その中に次のブロックが表示されます。

- **商品テンプレート** – これらのブロックには、商品の画像、タイトル、価格、カートに追加ボタンが表示されます。
- **ページネーション –** これらのブロックは、購入者がコレクション内をどのように移動するかを制御します。コレクションを1ページ分の商品のみ表示するように設定している場合、これらのコントロールは表示されません。
- **結果なし** – これらのブロックは、コレクションの「空の状態」を示します。つまり、購入者がページにアクセスした際に、設定したコレクションフィルターに一致する商品がない場合に表示される状態です。

これらのブロック要素はそれぞれ、複数のネストされた内部ブロックで構成されており、それぞれに独自の設定があり、必要に応じて削除できます。親ブロックと各内部ブロックの設定を確認するには、ブロックを選択し、設定サイドバーのブロックとスタイルの設定、およびブロックツールバーの設定を確認してください。ブロック設定の確認方法については、こちらのガイドをご覧ください。

既存のブロックの設定を変更するだけでなく、お好みのブロックを追加して商品テンプレートをさらにカスタマイズしたり、コレクションエリアにコンテンツを追加したり、レイアウトを変更したりすることも可能です。ぜひ自由に探索して、あなたの創造性がどこへ向かうのか、探してみてください！

**製品コレクション** ブロックには、製品の表示方法を制御するためのさまざまなレイアウト オプションとクエリ設定が用意されています。

＃＃＃＃ デフォルト

**デフォルト**クエリでは、現在のテンプレートに基づいて商品が表示され、買い物客はフィルタリングできます。**デフォルトの並べ替え条件**オプションは次のとおりです。

- マニュアル（メニューの順番 + 名前）
- 価格（高→安） / 価格（安→高）
- 売上（高→安）
- 評価（高→安）

**デフォルトクエリ** を使用するすべての製品コレクション ブロックは、この並べ替え順序に同期されます。

＃＃＃＃ カスタム

カスタム クエリを使用すると、ページのコンテキストに依存するのではなく (デフォルトのクエリ タイプの場合のように) 特定の条件に基づいて厳選された製品リストを表示できます。

このクエリ タイプを使用すると、**順序**、**ページあたりの製品数**設定**、**セール**、**在庫状況** フィルターを使用してカスタム コレクションを作成できます。

**並べ替え:**「カスタム」クエリタイプの場合、この設定でカスタムコレクション内の商品の並べ替え方法を選択できます。利用可能な並べ替えオプションは以下のとおりです。

- 商品名のアルファベット順（A → Z または Z → A）
- 新しい順 / 古い順
- 価格（高→安） / 価格（安→高）
- 売上（高→安） / 売上（安→高）
- 評価（高→安） / 評価（低→高）
- 手動（メニュー順 + 商品名）
- ランダム：ページが読み込まれるたびに商品の順序がシャッフルされ、訪れるたびに新鮮な体験を提供します。

**ページあたりの商品数**: ブロック内に一度に表示される商品数を指定します。ブロックにページ区切りが含まれていない場合、ここで指定した商品数のみが表示されます。

**セール中の製品のみを表示:** このオプションをオンにすると、現在セール中の製品のみが表示されます。

**在庫状況**: 製品コレクションに含める在庫状況を選択します。

レイアウト設定を使用すると、製品コレクション ブロック内での製品の配置方法を制御できます。

- **スタック**: 商品を縦に並べたリスト形式で表示します。
- **グリッド**: 商品を複数の列を持つグリッド形式で表示します。
- **カルーセル**: 商品を横向きのスライダー形式で表示します。ランディングページやおすすめセクションなど、視覚的な要素を重視したレイアウトに最適です。

商品コレクションブロックを使ってショップページをカスタマイズすると、商品の表示方法に様々なオプションが生まれます。ここでは、いくつかの重要な基本事項をご紹介します。レイアウトブロックを使って構造を変更したり、ブランドに合ったスタイルオプションを選択したりすることで、さらに魅力的で魅力的なデザインを作成できます。

製品コレクションに製品を 1 行だけ表示するように設定するには:

1. コレクションの**サイドバーブロック設定**で「グリッド」レイアウトオプションを選択し、商品行の列数を入力します。「レスポンシブ」トグルは**チェックなし**のままにしておきます。
2. コレクションの**ブロックツールバー**で「表示設定」アイコンをクリックし、「1ページあたりのアイテム数」を選択した列数に設定し、「最大表示ページ数」を**1**に設定します。

これで、製品コレクションが、選択した列の数だけ 1 行に表示されます。

特定のデザインを念頭に置いている場合は、製品コレクションを現在のページまたはテンプレートのクエリとフィルターに一致させるのではなく、特定のフィルターされた製品セットを表示したい場合があります。

製品コレクションにカスタムフィルターされた製品セットを表示するには:

1. コレクションの[サイドバーブロック設定](https://woocommerce.com/document/woocommerce-store-editing/blocks/#block-settings)で、「現在のクエリと同期」トグルを無効のままにします。「フィルター」セクションが表示されます。
2. サイドバーブロック設定の「フィルター」セクションで、3つのドットメニューをクリックし、クリックしてさまざまなフィルターオプションを切り替えます。
- リスト内のフィルターをクリックしてオンにすると、そのフィルターとそのオプションがブロック設定サイドバーの「フィルター」セクションに表示されます。
  
3. 選択したフィルターを使用して、このコレクションに表示するカテゴリ、タグ、属性、または組み合わせを選択します。

これは、ページ上の2つ目の商品コレクションをバナーとして活用し、特定の商品を買い物客におすすめするという素晴らしい方法です。可能性はほぼ無限大です！
