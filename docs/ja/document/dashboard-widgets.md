WordPressダッシュボードは通常、ログイン後に最初に表示されます。WooCommerceを有効にすると、ストアの概要を確認できるダッシュボードウィジェットが2つ追加されます。複数のストアでWordPressマルチサイトを使用している場合は、メインサイトに3つ目のウィジェットが追加されます。

ダッシュボードに表示されるウィジェットをカスタマイズするには、画面右上の「**画面オプション**」を選択して**画面要素**パネルを開きます。ここで表示されるオプションは、インストールしているプラ​​グイン、現在開いているページ、マルチサイトが有効かどうか、さらにはご利用のホスティング会社によって異なります。

WooCommerce セットアップウィザードを完了していない場合は、ダッシュボードに WooCommerce セットアップウィジェットが表示され、ストアに必要な設定を行うためのリマインダーとなります。WooCommerce セットアップウィザードが完了すると、WooCommerce ステータスウィジェットと WooCommerce 最近のレビューウィジェットが表示され、画面オプションパネルで利用できるようになります。

**WooCommerce Status** ウィジェットを使えば、ストアの概要を素早く確認できます。まず、当月の総売上高が表示され、次に売れ筋商品が表示されます。最後に、注文状況と在庫状況がグリッド表示されます。

処理待ちまたは保留中の注文があるかどうかが表示されます。また、在庫が少なくなっている商品や在庫切れの商品の数も表示されます。

**注記：**

[WooCommerce セットアップウィザード](https://woocommerce.com/document/woocommerce-setup-wizard/)が完了していない場合、**WooCommerce ステータス** ウィジェットは表示されません。代わりに、**WooCommerce セットアップ** ウィジェットが表示されます。

**WooCommerce Recent Reviews** ウィジェットには、ストアの最新のレビューが表示されます。

各レビューエントリには次の情報が表示されます。

- レビュー担当者の[Gravatar](http://en.gravatar.com/support/what-is-gravatar/)（Gravatarをお持ちの場合）。Gravatarをお持ちでない場合は、**設定 > ディスカッション**で選択した**デフォルトのアバター**が表示されます。
- 製品リンク
- レビュー担当者の名前
- レビューの抜粋
- 5つ星評価

WooCommerce Network Ordersウィジェットは、ネットワーク内のすべてのサイトからの注文情報を提​​供します。各注文番号は、特定のサブサイトの注文詳細画面にリンクされています。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
