ストア編集機能を使用すると、**商品カタログ**テンプレートを編集し、サイトエディター内のブロックを使用してレイアウトとスタイルをカスタマイズすることで、思い通りのショップページを作成できます。このドキュメントでは、WooCommerceに含まれるカタログページの管理とカスタマイズに使用できるテンプレートとブロックについて説明します。

WooCommerce は、メインのショップページ以外にも、状況に応じて異なるテンプレートで商品カタログを表示します。**カテゴリー**、**タグ**、**属性** 別に商品を表示するためのテンプレートが組み込まれています。

ブロックテーマを使用すると、ストアの多様なニーズに合わせて、様々な商品アーカイブをカスタマイズできます。また、ノーコードエディターを使用して、特定の属性、カテゴリ、タグ用のテンプレートを追加作成することもできます。

WooCommerceには、ストアでの商品の表示方法を管理するためのテンプレートが含まれています。選択したテーマによっては、これらのテンプレートが上書きされたり、テーマ固有のバージョンが追加されたりする場合がありますが、デザインニーズに合わせていつでも編集できます。[テンプレートへのアクセス方法の詳細](https://woocommerce.com/document/woocommerce-store-editing/templates/)をご覧ください。

デフォルトでは、ストアの製品カタログの表示は次のテンプレートによって制御されます。

- **商品カタログ** は、**WooCommerce > 設定 > 商品 > 一般** で設定されたメインショップページの表示を制御します。
- **カテゴリ別商品** は、ショップの商品カテゴリアーカイブページの外観を制御します。
- **タグ別商品** は、ショップの商品タグアーカイブページの外観を制御します。
- **属性別商品** は、ショップの商品属性アーカイブページの外観を制御します。
- **商品検索結果** は、ショップの商品検索結果ページの外観を制御します。

**注:** 異なる種類の商品アーカイブページ（タグ、カテゴリ、属性など）はそれぞれ異なるテンプレートで管理されているため、**すべての**商品アーカイブテンプレートを一度に編集することはできません。ただし、すべての商品アーカイブテンプレートで**同じデザインを複製**したい場合は、可能です。1. テンプレートをカスタマイズします。2. デザインを[テンプレートパーツまたはパターン](https://woocommerce.com/document/woocommerce-store-editing/patterns-template-parts/)として保存します。3. このテンプレートパーツまたはパターンを他のテンプレートに挿入すると、すぐに同じ外観になります。

以下のWooCommerceブロックは、ストア全体にカタログを表示するために中心的な役割を果たします。様々なデザインやユースケースに合わせてカスタマイズできます。他にも商品表示ブロックはいくつかありますが、そのほとんどは[非推奨](https://developer.woocommerce.com/2024/02/22/announceing-the-product-collection-block/)となり、より堅牢でカスタマイズ性の高い商品コレクションブロックに置き換えられています。

### 製品コレクションブロック

この**多目的ブロック**には、多数の既成コレクションが用意されています。また、カスタムコレクションを作成して、お好みの方法で商品を表示することもできます。[商品コレクションブロックの詳細](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/)

### 製品フィルターブロック

商品カタログページでは、複数のフィルタリングオプションをご利用いただけます。属性、評価、価格、在庫状況などでフィルタリングできるように設定できます。[商品フィルターブロックの詳細](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/filter-products/)

より早く開始できるように、グリッド、行などのさまざまな構成で製品コレクションを表示する複数の事前構築済みパターンから選択できます。[パターンの詳細](https://woocommerce.com/document/woocommerce-store-editing/patterns-template-parts/)。

上記のデフォルトテンプレートをカスタマイズすることに加えて、ストア編集を使用すると、特定の製品カテゴリの**カスタムテンプレートを作成**することが簡単になります。

カスタム テンプレートを作成するには:

1. **「外観」>「エディター」** でサイトエディターを開きます。
2. **「テンプレート」** をクリックし、プラス記号 (+) をクリックして新しいテンプレートを追加します。
3. テンプレートを適用する商品分類に応じて、**「商品カテゴリー」** または **「商品タグ」** オプションを選択します。
4. この新しいテンプレートを適用する商品カテゴリーまたは商品タグを選択します。
5. あらかじめ用意されたパターンを使用する場合は、プロンプトが表示されたらパターンを選択します。
6. 特定の商品カテゴリーまたはタグのニーズに合わせて、カテゴリー固有のテンプレートを編集します。
