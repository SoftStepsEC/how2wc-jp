WooCommerce を有効化すると、WordPress 管理ダッシュボードに 4 つのメインメニュー項目が追加されます。これらは、**WooCommerce**、**製品**、**アナリティクス**、**マーケティング** です。

WooCommerceのトップレベルにあるメインメニュー項目は、ストアで最もよく使う機能への入り口です。ホームのダッシュボード概要、注文と顧客へのリンク、設定、ストアのステータス、WooCommerce.comで購入した拡張機能などが含まれます。

**ホーム** をクリックすると、WooCommerce ダッシュボードとも呼ばれるホーム画面が表示されます。ここから、ストアのパフォーマンスを常に把握したり、ビジネスの成長に役立つヒントや分析情報を入手したり、ストア管理の重要な機能に素早くアクセスしたりできます。WooCommerce のホーム画面について詳しくは、下のボタンをクリックしてください。

「注文」画面では、既存の注文の確認と管理、または手動で新しい注文を追加できます。注文管理の詳細については、下のボタンを選択してください。

顧客セクションでは、顧客レポートをご利用いただけます。これは、要件に応じて検索・フィルタリングし、関連する顧客データを表示できるリストです。

レポートセクションでは、注文、顧客、在庫、税金に関する分析情報を含む従来のレポートを提供しています。[WooCommerce Analytics](https://woocommerce.com/document/woocommerce-analytics/) は、より強力で最新の代替手段であり、以前のレポートが廃止されるまでは、これらのレポートと共存します。

設定画面では、一般、製品、税金、配送、支払い、アカウントとプライバシー、電子メール、統合、ページ設定、REST API などの詳細設定を含む、WooCommerce の高度な設定の大部分を構成します。

ステータスページは、ご自身でトラブルシューティングを行う場合や、サポートに問い合わせる際に便利です。システムステータスレポートをダウンロードして確認するか、サポートチケットに添付してください。このレポートには、設定の全体的なスナップショットと潜在的な競合状況が表示されます。

拡張機能セクションでは、WooCommerce.com の無料/有料プラグインや拡張機能を検索して、サイトに機能や特徴を追加できます。**「探す」**、**「参照」**、または**「テーマ」** タブから、ストアを強化する拡張機能を見つけ、「マイサブスクリプション」** タブからサブスクリプションを管理できます。

## 製品メニュー

製品のトップレベル メニューには、新しい製品の表示と追加、製品カテゴリとタグの設定、製品属性の管理など、製品に関連するセクションがあります。

**すべての商品** は、ストア内の商品（公開済み、下書き、ゴミ箱内）のリストです。この画面から新しい商品を手動で追加したり、個別または一括編集したり、CSV経由で商品をインポートまたはエクスポートしたりできます。

**新規追加** メニュー項目は、新しい製品を手動で追加するための **新製品の追加** 画面へのショートカットです。

ブランドは、製品をブランド別に整理するのに役立つ組み込みの分類法です。

これら 3 つのメニュー項目は、それぞれカテゴリ、タグ、属性の管理画面にリンクしています。

[レビュー] をクリックすると、**製品レビュー** セクションに移動し、サイトの訪問者が使用または購入した製品について残したレビューを管理できます。

**分析**メニューには、商品、収益、注文、クーポン、税金、ダウンロード、在庫など、あらゆる情報を網羅したストアのパフォーマンスを分析できるさまざまなレポートが用意されています。

マーケティング メニューでは、ストアのプロモーションと成長に不可欠なマーケティング機能にアクセスできます。

このセクションは**マーケティング ハブ**とも呼ばれ、ストアを宣伝して売上を伸ばすためのツールと知識を提供します。

「クーポン」メニュー項目は「クーポン」画面にリンクしており、そこから新しいクーポンを追加したり、既存のクーポンを管理したりできます。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
