グローバル時代において、一つの地域、一つの言語、一つの通貨のみで販売する店舗オーナーはほとんどいません。そこには様々な要因が関わっており、以下ではそのうちのいくつかについて解説します。

地域に応じて異なる設定をしたいのが税金と配送だけであれば、そのためには WooCommerce コアを使用できます。

1. ストアのWP管理ダッシュボードで、*WooCommerce > 設定 > 税金 > 税金オプション* に移動します。
2. 「税金の計算基準」の横にある **顧客の配送先住所** を選択します。
3. 次に、*WooCommerce > 設定 > 税金 > 標準税率* に移動します。
4. 販売先の国の税率を追加します。

顧客が別の場所から購入する場合、税金も異なります。

同様に、[配送ゾーンの設定](https://woocommerce.com/document/setting-up-shipping-zones/)もできます。以下の例では、イギリスへの送料を10ポンド（GBP）の定額に設定し、オランダへの送料を送料無料に設定しています。

1. WP管理ダッシュボードで、*WooCommerce > 設定 > 配送 > 配送ゾーン*に移動します。
2. **ゾーンを追加**をクリックします。
3. **ゾーン名**、**地域**、**配送方法**を追加します。
4. イギリスの場合は**定額**、オランダの場合は**送料無料**を選択します。

これで、英国とオランダに異なる配送が設定されました。

プラットフォーム上で異なる通貨を提供したい場合は、[通貨換算ウィジェット](https://woocommerce.com/products/currency-converter-widget/)を使用して、顧客が好みの通貨を選択できるようにします。

すべての通貨は、主要通貨との相対関係に基づいて自動的に計算されます。例えば、主要通貨をGBPに設定し、その横にユーロと米ドルの表示を許可した場合、商品の価格をGBPで入力すると、その他の価格はすべて自動的に計算されます。換算された価格を手動で編集することはできません。

このウィジェットを使用するには、[ドキュメント](https://woocommerce.com/document/currency-converter-widget/)をご確認ください。ドキュメントのご確認後、お客様は通貨を変更できるようになります。

ストアを複数の言語で提供したい場合、**MultilingualPress** は最適な選択肢の一つです。まずは、MultilingualPress の機能と制限について見ていきましょう。

それは次のようになります:

- **複数のウェブサイトを[WordPressネットワーク](https://developer.wordpress.org/advanced-administration/multisite/create-network/)上で商品レベルとページレベルで接続**できます。例えば、英国ストアの「カート」ページと米国ストアの「カート」ページをコンテンツで接続できます。
- **すべての言語のコンテンツを1か所で編集**できます。例えば、英語ストアで商品を編集しているときに、オランダ語の商品の翻訳を追加できます。これにより、オランダ語ストアの商品も更新されます。
- **商品レベルとページレベルで言語を簡単に切り替え**できます。製品Xを英語で表示している場合、オランダの国旗をクリックすると、製品Xのオランダ語版にアクセスできます。（この例では国旗を追加しましたが、デフォルトではシンプルなリンクが表示されます。）
- **設定を一元管理する。** WordPressネットワークでは、ユーザー、テーマ、プラグインなど、いくつかの設定をマルチサイトレベルで管理できます。また、一部の設定を複数のウェブサイトで一括管理できます。MultilingualPressを使用すると、翻訳へのクイックリンクを追加したり、顧客のブラウザ言語でストアへのリダイレクトを設定したりできます。
- **WooCommerceを個別に設定する。** 各ストア内で商品を接続できますが、通貨、税金、配送などについては個別に設定できます。WooCommerceの各設定は個別に設定できます。
- **パフォーマンスの低下を回避する。** 同じウェブサイトに同じ製品の複数の言語バージョンを追加すると、パフォーマンスが低下する可能性があります。MultilingualPressでは、異なる言語バージョンが別々のウェブサイトに配置されるため、このような問題は発生しません。
- **特定の言語の SEO を最適化します。** MultilingualPress は言語ごとに個別の Web サイトに焦点を当て、Web サイトの検索エンジン最適化に役立ちます。

以下のことはありません:

- **通貨換算を自動化。** 別々の通貨を設定できますが、商品の価格は手動で変更する必要があります。
- **在庫を一元管理。** 商品の各バージョンは引き続き個別に管理されます。言語は連携されていますが、在庫は連携されていません。

MultilingualPressは、豊富なオプションを備えた多機能な拡張機能です。以下では、概要を簡単にご紹介します。

#### 1. ネットワークを作成する

1. **WordPressネットワークを設定します。** MultilingualPressは[ネットワーク](https://developer.wordpress.org/advanced-administration/multisite/create-network/)でのみ動作します。そのため、まずネットワークを作成する必要があります。
2. **WooCommerceとMultilingualPressの両方を**ネットワークにインストールします。WordPress管理画面で、*「登録サイト」>「プラグイン」>「新規追加」*に移動します。そこからWooCommerceを検索してインストールします。次に、MultilingualPressについても同様の手順を実行します。
3. **MultilingualPressのみをネットワークで有効化します。**これにより、ネットワーク上のすべてのサイトで拡張機能が有効になります。**WooCommerceではこの操作を行わないでください**。税金と送料の設定​​に問題が生じる可能性があります。MultilingualPressとWooCommerceをネットワークで有効化した場合、税金と送料はメインストアでのみ編集でき、セカンダリストアには反映されません。
4. *「マイサイト」>「設定」>「MultilingualPress」* に移動し、有効にする設定を選択します。**「カスタム投稿タイプ翻訳」** を有効にします。

#### 2. メインストアを設定する

1. **メインストアを作成します。** *「マイサイト」>「サイト」>「新規追加」* に進みます。すべての設定を完了します。これがメインストアになる場合は、**言語** 以外の MultilingualPress 設定は空白のままにしておきます。言語は選択する必要があります。
2. 次に、*「マイサイト」>「サイト」>「すべてのサイト」* に進みます。サイトのダッシュボードに移動します。
3. *「ダッシュボード」>「プラグイン」* に移動し、WooCommerce を有効化します。
4. [WooCommerce を設定](https://woocommerce.com/documentation/woocommerce/getting-started/) します。
5. [商品を追加](https://woocommerce.com/document/managing-products/) します。

設定を変更したり、後から製品を追加したりすることはいつでも可能ですが、MultilingualPress を使用する場合は、この拡張機能を使用すると、設定と製品をコピーし、製品をリンクするなど、メインサイトに基づいて 2 番目のサイトを構築できるため、できるだけ詳細に行うことをお勧めします。

#### 3. 別の言語で2つ目のストアを設定する

1. **2つ目のストアを作成します。** *「マイサイト」>「サイト」>「新しいサイトを追加」* に移動します。すべての設定を行います。メインサイトとの**関係** を有効にし、この新しいサイトが **サイトに基づいている** ことを選択します。ここでメインサイトを選択します。これにより、「ソースサイトでアクティブなすべてのプラグインを有効化」が可能になり、WooCommerce が有効化され、設定がコピーされます。
2. 2つ目のサイトの**ダッシュボード** に移動します。
3. 次に、*「ダッシュボード」>「WooCommerce」>「設定」* に移動します。そこから、2つ目のサイトを設定します。異なるロケール、税金、配送方法、支払い方法などを設定することもできます。
4. *「ダッシュボード」>「商品」* に移動すると、すべての商品がメインサイトからコピーされていることがわかります。
5. これで、各商品を**編集** できます。通貨を変更した場合は、少なくとも**価格** を確認してください。商品の**説明**はどちらのストアでも編集できます。例えば、下のスクリーンショットでは、異なるオランダ語翻訳が追加されています。
6. 2つの商品間の**関係**を変更することもできます。現在、英語ストアのFlying Ninjaはオランダ語版のFlying Ninjaと接続されていますが、これは変更できます。
7. **カスタマイズ**を選択します。次に、*ウィジェット > サイドバー > ウィジェットの追加 > 言語スイッチャー*に進みます。設定が完了すると、顧客は2つのストアを切り替えることができるようになります。

顧客は英語ストアから始めて商品にアクセスし、2 つの異なるストア間を行き来できるようになります。

どちらかのストアで商品をカートに追加すると、税金、配送、支払いに関するオプションは、ストアの設定方法によって異なります。

- ここでの英国ストアの例では、発送先は英国のみで、送料は一律5ポンド、英国の税金は20%、支払い方法はPayPalまたは銀行振込です。
- オランダストアの例では、発送先はベルギーとオランダの両方で、送料無料、オランダの税金は21%、支払い方法は銀行振込のみです。
