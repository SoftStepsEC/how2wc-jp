Security for WooCommerce protects your store from hackers and cyberattack. Designed specifically for WooCommerce, it helps safeguard your store against malicious traffic and vulnerabilities in real time.

- **Download** the .zip file from your [WooCommerce account](https://woocommerce.com/my-account/downloads/).
- Go to:**WordPress Admin > Plugins > Add New**> **Upload Plugin**
- **Choose File** for the file you downloaded.
- **Install Now** and **Activate Plugin**.

More information at: [Install and Activate Plugins/Extensions](https://woocommerce.com/document/managing-woocommerce-com-subscriptions/#section-2).

To set up your store with Security for WooCommerce:

1. Go to: **WooCommerce > Settings > Integration > Security for WooCommerce**
2. Enable Firewall On/Off button to activate the plugin.
3. Your website IP address will be loaded automatically in the next tab.
4. Select your location country in **Your current location/country** tab.

To block and allow audience from multiple countries:

1. Go to **Blocked countries list** tab.
2. Click **Select All** to check all the countries, uncheck your country of origin and keep unchecking all other countries you want to allow, keeping Ctrl or ⌘ key pressed while selecting from the list of countries.
3. Save changes by clicking **Save changes** button at the bottom of Security for WooCommerce settings.

![More information at: Install and Activate Plugins/Extensions](docs/images/cd0ed2872decd22efa278c7867843bc3.png "https://woocommerce.com/wp-content/uploads/2022/09/multiple-block-Security-for-WooCommerce-images.png?strip=all&w=704")

If your business serves only one country and you want to restrict access to audiences from that specific location, then:

1. Go to **Allow only traffic from this country** tab.
2. Select the country you want to allow audience for and save changes.

![More information at: Install and Activate Plugins/Extensions](docs/images/d2597acc90d8ab5386235e2fac25c17d.png "https://woocommerce.opmc.com.au/wp-content/uploads/2022/09/single-allow-Security-for-WooCommerce-images.png")

To manually whitelist IP addresses;

1. Navigate to **Whitelist IPs** tab.
2. Add the IP address to whitelist. Press **Tab**, **Enter** or **Comma ( , )** to add another IP address.

To manually backlist IP addresses:

1. Go to **Blacklisted IPs** tab.
2. Enter IP address to add to the blacklist. Press **Tab**, **Enter** or **Comma ( , )** to add another IP address.

When a visitor arrives from a country that the merchant is not offering business to, they will see the following message, “**Sorry, your country is not permitted to visit this website.**“

![More information at: Install and Activate Plugins/Extensions](docs/images/ee9d799f5411ccf6f3b2bff57106e1cf.png "https://woocommerce.com/wp-content/uploads/2022/09/sorry-not-permitted-opmc-security-for-woocommerce.png?strip=all&w=704")

Also, when customers come from allowed countries, on checkout they will be given option to ship products to only countries that merchant has authorised to offer service.

![More information at: Install and Activate Plugins/Extensions](docs/images/ec36287cc7b52d47424a405700c59a7d.png "https://woocommerce.com/wp-content/uploads/2022/09/Checkout-security-for-woocommerce.png?strip=all&w=704")

If you encounter an error, turn the Debug Log On/Off, save the changes, and reach out to us for assistance.
