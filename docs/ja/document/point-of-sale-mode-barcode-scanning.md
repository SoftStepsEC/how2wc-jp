WooCommerce モバイルアプリは、**販売時点管理 (POS)** モード使用時にバーコードスキャンをサポートしています。バーコードスキャンを使用すると、対応する外付けバーコードスキャナーを接続して商品のバーコードをスキャンし、POS モード時に商品を自動的にカートに追加できます。

バーコードのスキャンを開始するには、製品とバーコード スキャナーが適切に設定されていることを確認する必要があります。

**ご注意:** バーコードスキャンには[販売時点管理(POS)](https://woocommerce.com/document/woo-mobile-app-point-of-sale-mode/)モードが必要です。つまり、ここで概説されているWoo Mobile POSモードと互換性のあるデバイスを使用する必要があります。

**ご注意ください:** 注文と製品タブとは異なり、SKU は在庫管理を目的としているため、POS ではバーコードの SKU はサポートされていません。

製品をスキャン用に設定するには、スキャンする各製品にバーコード値を追加する必要があります。バーコード値を追加するには、以下の手順に従います。

1. WordPressウェブサイトで、**「商品」>**「すべての商品」**に移動します。
2. 設定する商品の下にある**「編集」**を選択し、**「商品詳細」**を開きます。
3. *「商品詳細」ページの*「商品データ」で、**「在庫」**を選択します。
4. GTIN、UPC、EAN、ISBNの各フィールドに、対応するバーコード値を入力します。
5. 商品を**「更新」**して変更を保存します。

Woo モバイル アプリから直接製品にバーコード値を追加することもできます。

1. *Woo モバイルアプリ* で、下部メニューバーの **商品** タブを選択します。
2. バーコード値を追加する **商品** をタップして選択します。
3. *商品詳細* ボタンのリストで **在庫** を選択します。
4. GTIN、UPC、EAN、および ISBN フィールドに、対応するバーコード値を入力します。
5. 商品を **更新** して変更を保存します。

**ご注意：** Woo Mobileアプリは、スキャナーから提供された情報を正確に検索します。つまり、商品のGTIN、UPC、EAN、ISBNの各フィールドのすべての文字が含まれている必要があります。チェックディジットと国コードも含まれます。コードの最小桁数は6桁ですが、UPC-Aの標準桁数は12桁、GTIN/EAN/ISBNは13桁です。

Woo Mobileアプリは、**Bluetooth HID**（ヒューマンインターフェースデバイス）を使用して接続するほとんどのバーコードスキャナーをサポートしています。HIDは、スキャナーがキーボード入力を非常に高速にエミュレートするモードであり、これによりアプリは幅広いスキャナーをサポートできます。

互換性を確保するために直接統合テスト済みの以下のスキャナーをお勧めします。

- [Netum 1228BC](https://www.netum.net/products/nt-1228bc-bluetooth-ccd-barcode-scanner-read-1d-code)
- [Star Micronics BSH-20B](https://href.li/?https://star-emea.com/products/bsh-20b/)
- [Tera 1200 2D](https://tera-digital.com/products/1200-2d-portable-barcode-scanner-wholesale)

推奨されるスキャナーについては、以下の簡素化されたセットアップ プロセスに従ってスキャナーを接続し、構成を確認できます。

1. [POSモード](https://woocommerce.com/document/woo-mobile-app-point-of-sale-mode/) で、左下にある**メニュー**（3つの点）ボタンを選択します。次に、**バーコードスキャナーの設定** を選択します。
2. 設定するスキャナーを選択するためのメニューが表示されます。以下の該当セクションをご参照ください。

**注:** リストにないスキャナーの場合は、[**その他**] を選択し、[以下の手順](https://woocommerce.com/document/point-of-sale-mode-barcode-scanning/#h-connecting-other-scanners)に従って互換性のあるスキャナーを接続します。

1. 表示されたリストからスキャナーを選択したら、スキャナーの電源を入れ、表示されたコードをスキャンします。スキャナーに設定コードのスキャン完了が表示されたら、「次へ」を選択します。
2. 次に、スキャナーのペアリングを促す画面が表示されます。「デバイス設定に移動」を選択すると、自動的にリダイレクトされ、スキャナーをBluetoothで接続できます。Bluetooth接続に関するサポートガイドは、各オペレーティングシステムごとに以下のリンクからご覧いただけます。
- [iOSまたはiPadOSでBluetoothデバイスを接続する](https://support.apple.com/en-us/105108)
- [AndroidでBluetoothデバイスを接続する](https://support.google.com/android/answer/9417604?hl=en&visit_id=638882717196697984-1314012052&rd=1)
  
3. デバイスが Bluetooth 経由で正常にペアリングされたら、WooCommerce アプリに戻り、「**次へ**」を選択します。
4. 提供されているサンプルバーコードをスキャンして、バーコードスキャン機能をテストします。

**注:** アプリが10秒以内にバーコードをスキャンしたことを検出しない場合、データが検出されなかったことを通知します。スキャン後も画面が変わらない場合は、セットアッププロセスを完了するか、このガイドのトラブルシューティングセクションを参照してください。

1. 成功すると、「**スキャナーがセットアップされました!**」と表示され、セットアップ モーダルを閉じて、外部スキャナーを使用して製品のバーコードのスキャンを開始できます。

#### 他のスキャナーを接続する

多くのスキャナーはデフォルトで互換モードに設定されています。スキャナーのマニュアルを参照することで、スキャナーが正しく設定されていることを確認できます。スキャナーはBluetooth HIDを使用し、各コードの末尾にキャリッジリターン文字とラインフィード文字（`/r` または `/r/n`）を送信するように設定する必要があります。

スキャナーの設定が完了したら、Bluetoothを使用してデバイスに接続します。お使いのオペレーティングシステムに応じたBluetoothデバイスの接続に関するヘルプガイドは以下をご覧ください。

- [iOS または iPadOS で Bluetooth デバイスを接続します。](https://support.apple.com/ja-jp/105108)
- [Android で Bluetooth デバイスを接続します。](https://support.google.com/android/answer/9417604?hl=ja&visit_id=638882717196697984-1314012052&rd=1)

設定が完了したら、iOS または Android の Bluetooth 設定でタブレットに接続します。

アプリ以外では、メモアプリやその他のテキスト フィールドでスキャンしてスキャナーをテストできます。スキャナーはキーボードとして動作するため、バーコード番号がメモに入力されます。

設定が完了したら、商品のスキャンを開始するには、まずPOSモードに入る必要があります。POSモードに入ると、バーコードスキャナーを使って商品をスキャンできます。

スキャンが成功すると、対応する製品が**カート**に追加されます。

スキャンに失敗した場合、バーコード番号は「不明なスキャン済みアイテム」というエラーとともにカートに追加されます。このアイテムは、横にある**ゴミ箱**アイコンを選択することで**カート**から削除できます。

Androidはバーコードスキャナーを外部キーボードとして認識するため、一部のソフトウェアキーボードが自動的に非表示になる場合があります。画面上のキーボードが表示されない場合は、以下のトラブルシューティング手順をお試しください。

1. スキャナーが接続されたら、Androidの設定で**オンスクリーンキーボードを表示** を有効にします。
2. キーボードアプリにも独自の設定がある場合があります。キーボードアプリの設定を確認してください（Gboard、Samsungキーボードなど）。
3. **「物理キーボード接続時にオンスクリーンキーボードを表示する」** オプションを有効にします。

この設定はキーボード アプリとデバイスによって異なります。

スキャナーが正しく動作しているかどうかを確認するには、メモアプリまたはドキュメントアプリを開いてバーコードをスキャンすることをお勧めします。製品のバーコード値の後に、復帰文字と改行文字（`/r` または `/r/n`）が表示されます。

いいえ、デバイスのカメラを使用してバーコードをスキャンすることはできません。ハードウェア スキャナーのみをサポートしています。

HIDスキャナーでスキャンすると、外付けキーボードがエミュレートされます。スキャナーを使用すると、iOSまたはAndroidが外付けキーボードを使用していると判断し、ソフトウェアキーボードを非表示にすることがあります。iOSまたはAndroidの画面に表示されるキーボードボタンを使用してキーボードを再度有効にすることで、キーボードを再び表示できます。

一部のスキャナーでは、トリガーを素早く 2 回引くことでソフトウェア キーボードを復元することもできます。

#### Android のトラブルシューティング

Androidはバーコードスキャナーを外部キーボードとして認識するため、一部のソフトウェアキーボードが自動的に非表示になる場合があります。画面上のキーボードが表示されない場合は、以下のトラブルシューティング手順をお試しください。

1. スキャナーが接続されたら、デバイスの設定で「オンスクリーンキーボードを表示」オプションを見つけて有効にします。*設定*アプリでは通常、「ハブモード」>「概要」>「タブレットを検索」にあります。
2. GboardやSamsungキーボードなどのサードパーティ製キーボードをご利用の場合は、独自の設定がある場合があります。キーボードの設定を確認するか、キーボードの開発元に直接サポートを依頼することをお勧めします。
3. デバイスの設定で、「物理キーボードが接続されたときにオンスクリーンキーボードを表示」オプションを見つけて有効にします。この設定の場所はデバイスによって異なります。具体的な情報については、デバイスの製造元にお問い合わせください。

この設定はキーボード アプリとデバイスによって異なります。

スキャナーはキーボードとして動作するので、ハードウェアキーボードレイアウトを米国英語に設定すると便利です。数字のみのバーコードの場合は通常必須ではありませんが（推奨）、一部の問題の解決に役立つ場合があります。
