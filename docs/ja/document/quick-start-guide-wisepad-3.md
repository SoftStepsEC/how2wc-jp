WisePad 3 は英国とカナダの販売店でサポートされています。

- WisePad 3 カードリーダー x 1
- USB-C充電ケーブル x 1
- クイックスタートガイド

このデバイスはiOSとAndroidをサポートしています。

1. WooCommerceストアに[WooCommerce](https://woocommerce.com/start/?nuxentrysource=docs_text_cta)と[WooPayments](https://woocommerce.com/products/woocommerce-payments/)がインストールされ、有効化されていることを確認してください。
2. [iOS](https://apps.apple.com/us/app/id1389130815)または[Android](https://play.google.com/store/apps/details?id=com.woocommerce.android)用の[WooCommerceモバイルアプリ](https://woocommerce.com/mobile/)をダウンロードしてください。既にアプリがインストールされている場合は、最新バージョンにアップデートされていることを確認してください。
3. [iOS](https://woocommerce.com/document/woocommerce-ios/)または[Android](https://woocommerce.com/document/android/)アプリの指示に従って登録またはログインしてください。

付属の充電ケーブルを使ってカードリーダーを約2～3時間充電してください。初めて使用する前に充電することで、カードリーダーの損傷を防ぐことができます。デバイス画面の左上にバッテリー残量インジケーターアイコンが表示され、充電状況を確認できます。

**注:** 通常の使用状況では、リーダーは1日に1回充電する必要があります。完全に充電されたリーダーは、1回の充電で約600～800件の取引を実行できます。

充電後、リーダーのPINパッドの右上にある電源ボタンを短く押してデバイスの電源を入れます。ディスプレイが点灯し、デバイスのスプラッシュスクリーンが表示されます。数秒間操作がないとディスプレイは暗くなります。5分以上操作がなく、アプリケーションから切断されている場合は、ビープ音が鳴り、電源が切れます。WisePad 3を手動でオフにするには、電源ボタンを長押しし、LEDディスプレイに「電源をオフにしますか？」と表示されたら、緑色のEnterボタンを押して確定します。

**始める前に:**デバイスのBluetooth設定でカードリーダーをペアリングしないでください。WooCommerceモバイルアプリが自動的に接続を管理します。

お使いのデバイスでBluetoothが有効になっていることを確認してください。WooCommerceモバイルアプリを開きます。**設定（歯車アイコン）> 対面決済 > カードリーダーの管理**に移動します。

「カードリーダーを接続」をクリックします。プロンプトが表示されたら、アプリの使用中に位置情報の取得を許可してください。

スマートフォン/タブレットの画面にカードリーダーのシリアル番号が表示されたら、カードリーダーを選択してください。次に、ペアリングPINを使用してデバイスとWisePad 3をペアリングします。これでWisePad 3カードリーダーが接続されました。決済を受け付けるには、WooPaymentsアカウントの「テストモード」が無効になっていることを確認してください。リーダーはスタンバイ状態の間もiOSまたはAndroidデバイスに接続したままになり、アクティビティを再開すると自動的にスタンバイモードを終了します。WisePad 3を手動でオフにするには、電源ボタンを長押しし、LEDディスプレイに「電源をオフにしますか？」と表示されたら、緑色のEnterボタンを押して確定します。

**注:**2025年10月13日以降、新しいファームウェアアップデートには追加のペアリング手順が必要になる場合があります。該当する場合、カードリーダーとモバイルアプリの両方に6桁のコードが表示されます。これらの番号が一致する場合は、両方のデバイスでペアリングプロセスを完了することを確認してください。

**注:**オンラインで注文した商品を、カーブサイドピックアップや店舗受け取りなど、対面で支払いを受けるためにカードリーダーを使用する場合は、[ドキュメント](https://woocommerce.com/document/getting-started-with-in-person-payments-with-woocommerce-payments/#section-2)の指示に従って、ストアのWP Admin設定にいくつか変更を加える必要があります。これには、店舗受け取りゾーンの設定、代金引換の有効化、商品がこのゾーンに設定されていることを確認することが含まれます。

お客様がオンラインで注文した商品を店舗受け取りで受け取るには、WooCommerceモバイルアプリで該当の注文に移動し、「支払いを受け取る」をクリックして取引を完了します。その後、クレジットカードをタップまたは挿入して支払いを承認します。PINベースの支払いの場合は、カードを挿入し、WisePad 3のPINパッドを使用して支払いを完了します。

外出先ですぐにお支払いが必要な場合は、注文作成機能をご利用ください。モバイルアプリの「注文」タブに移動し、「+」記号をクリックして注文を作成します。その後、カードリーダーを使ってお支払いを受け取ります。

NFC（近距離無線通信）は、対応デバイス間の近距離通信を可能にする機能です。対面決済の場合、顧客のNFC対応銀行カードとカードリーダー間の通信を指します。

この方法でお支払いいただくと、成功または失敗がビープ音でお知らせします。カード情報を読み取ると、リーダーの4つのLEDライトがすべて緑色に点灯します。カードの読み取りが成功すると、ビープ音が1回鳴ります。ビープ音が2回鳴った場合はエラーが発生していますので、再試行してください。

WooCommerce モバイル アプリが最新バージョンに更新されていること、WooCommerce と WooPayments が最新であること、モバイル アプリにログインしていることを確認してください。

**注:**「カードリーダーの管理」オプションは次の場合にのみ表示されます。

- 注文ステータスは「保留中」、「保留中」、または「処理中」です。
- 支払い方法は「代金引換」、「woocommerce_payments」、または「なし」です。
- ストアの住所はカナダまたは英国の住所に設定されています。
- 通貨は「CAD」または「GBP」です。
- 注文は支払われていません。
- 注文は返金されていません。
- 注文に定期購入商品がありません。

- 電源ボタンを一度押して電源をオフにし、もう一度押して電源をオンにして、リーダーを再起動してください。
- 問題が解決しない場合は、WooCommerce モバイルアプリを強制終了して再度開くか、デバイスとリーダーを再起動してください。
- リーダーが充電されており、他のデバイスに接続されていないことを確認してください。
- リーダーがデバイスの Bluetooth 設定で直接ペアリングされていないことを確認してください。以前にペアリングしたデバイスの Bluetooth 設定に移動し、「デバイスの削除」を選択してください。
- リーダーが WooCommerce モバイルアプリの範囲内（通常 3～6 メートル）にあることを確認してください。

- カードリーダーの電源をオフにしてから、再度オンにしてください。初期設定が完了している場合は、カードリーダーはスマートフォンまたはタブレットに自動的に再接続されます。
- バッテリー残量が少ない可能性があります。USBケーブルを使用してリーダーを充電してから、再試行してください。
- デバイスまたはスマートフォン/タブレットがWi-Fiの受信範囲内にあることを確認してください。
- WooCommerce > 設定 > 支払い の WooPayments 拡張機能設定で、WooPayments アカウントが設定され、接続されていることを確認してください。
- (カードを挿入) リーダーに挿入する際に、カードのチップが正しい方向を向いていることを確認してください。
- (カードをタップ) カードが適切な支払い方法をサポートしているかどうかを確認してください（例：タップ機能の場合は、カードに適切なシンボルが付いている必要があります）。カードは、財布に入れずに、タップマークから1.5インチ（4cm）以内の範囲に置く必要があります。

- 電源ボタンを押してリーダーの電源を再度入れてください。リーダーはスマートフォンまたはタブレットに自動的に再接続されます。
- バッテリー残量が少ない可能性があります。USBケーブルで充電してから、もう一度お試しください。
- スマートフォン／タブレットがカードリーダーから数メートル以内にあることを確認してください。

追加のトラブルシューティング サポートについては、[WooPayments を使用した対面支払いの開始](https://woocommerce.com/document/getting-started-with-in-person-payments-with-woocommerce-payments/) または [Stripe WooCommerce 拡張機能を使用した対面支払いの受け入れ](https://woocommerce.com/document/stripe/admin-experience/in-person-payments/) ガイドを参照してください。

Jetpackプラグインを使用してアプリをサイトに接続している場合は、Jetpack接続がアクティブで正常に動作していることを確認してください。[既知の問題](https://jetpack.com/support/getting-started-with-jetpack/known-issues/)を確認するか、[サイトの再接続](https://jetpack.com/support/reconnecting-reinstalling-jetpack/)をお試しください。

それでも問題が解決しない場合は、アプリ内から「メニュー」>「設定」>「ヘルプとサポート」>「サポートに問い合わせ」にアクセスし、お問い合わせフォームにご記入の上、サポート チームにお問い合わせください。
