Jetpackプラグインがインストールされ、サイトに接続されている必要があります。Jetpackを初めてご利用になる場合は、[こちらのサポートガイド](https://jetpack.com/support/getting-started-with-jetpack/)を参照して、使い始めてください。

Jetpack [12.3](https://github.com/Automattic/jetpack/blob/trunk/projects/plugins/jetpack/CHANGELOG.md#123---2023-07-05)以降では、[Jetpack 機能モジュール ページ](https://jetpack.com/support/control-jetpack-features-on-one-page/)で Blaze を有効化する必要があります。

**または** [Blaze for WooCommerce](https://woocommerce.com/products/blaze-ads/) 拡張機能をインストールする必要があります。

## コンテンツを盛り上げる

Blaze キャンペーンを設定するには、次の手順に従います。

1. メニュータブをタップし、**Blaze**をクリックします。**キャンペーンを開始**ボタンをクリックします。

2. 宣伝したい商品を選択し、「**宣伝**」ボタンをクリックします。

3. 次のページでは、必要に応じて広告を編集できます。広告は、ネットワーク全体でプロモーションされるBlazeキャンペーンの見た目を決定します。画像、商品名、説明を変更できます。ただし、画像が機能するには、**500 x 500ピクセル**以上のサイズが必要ですのでご注意ください。

- **画像の変更:**広告用の画像をアップロードします。この画像はデフォルトで投稿のアイキャッチ画像として表示されますが、別の画像をアップロードすることもできます。また、ここで画像をトリミングすることもできます。
- **キャッチフレーズ**: 画像のすぐ下に表示される、太字ではないテキストです。商品名など、キャッチフレーズを入力できます。32文字を超える場合は切り捨てられます。広告の内容を簡単に説明するために使用してください。
- **説明**: 画像とタイトルの下に表示される、商品に関する短い説明文です。最大140文字まで入力できます。このテキストはタイトルを補足し、視聴者が広告をクリックしてコンテンツにアクセスした際に目にする内容をより詳しく伝えるために使用することをお勧めします。また、視聴者の注目を集めるためにも使用できます。

4. 次は**詳細**セッションです。**予算**タブをクリックすると、Blazeキャンペーンに費やす金額を管理できます。

5. オーディエンス セッションでは、次の項目を調整して、広告を表示するユーザーを選択できます。

- **言語**: 対象ユーザーが話すターゲット言語（複数可）を選択します。複数のターゲット言語を選択でき、「すべての言語」を選択すると、利用可能なすべての言語をターゲットにできます。
- **デバイス**: モバイルデバイスまたはパソコン（デスクトップとノートパソコンの両方）を選択できます。
- **興味関心**: ドロップダウンから対象ユーザーの興味関心を選択します。「すべてのトピック」を選択すると、興味関心に関係なく、すべての潜在的な視聴者に投稿が表示されます。
- **地域**: 投稿をプロモーションする地域を選択します。対象ユーザーを絞り込むことも、「すべての地域」を選択すると、場所に関係なくすべての潜在的な視聴者に投稿が表示されます。この画面で設定を調整すると、広告プレビューに選択内容が反映され、キャンペーンの推定リーチ（広告の表示回数）が表示されます。

6. **詳細を確認**ボタンをクリックします。

7. 支払いの詳細を確認し、キャンペーンの送信をクリックします。

キャンペーンを停止したい場合は、次の手順に従ってください。

1. [wordpress.com/advertising](http://wordpress.com/advertising)**にアクセスし、「キャンペーン」タブをクリックします。
2. 停止したいキャンペーンの右側にある下向き矢印をクリックします。
3. 右下にある「キャンペーンをキャンセル」リンクをクリックします。

キャンペーンがキャンセルされると、実行が停止され、再開することはできません。

すでに送信済みのキャンペーンを編集する必要がある場合は、そのキャンペーンをキャンセルし、必要な更新を加えた新しいキャンペーンを作成して、承認のために送信することができます。

Jetpackプラグインを使用してアプリをサイトに接続している場合は、Jetpack接続がアクティブで正常に動作していることを確認してください。[既知の問題](https://jetpack.com/support/getting-started-with-jetpack/known-issues/)を確認するか、[サイトの再接続](https://jetpack.com/support/reconnecting-reinstalling-jetpack/)をお試しください。

それでも問題が解決しない場合は、アプリ内から「メニュー」>「設定」>「ヘルプとサポート」>「サポートに問い合わせ」にアクセスし、お問い合わせフォームにご記入の上、サポート チームにお問い合わせください。
