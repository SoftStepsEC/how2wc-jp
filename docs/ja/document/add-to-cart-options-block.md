「オプション付きでカートに追加」ブロックには、「カートに追加」ボタンと数量選択ボタンが表示され、商品タイプによっては追加の選択項目も表示されます。このブロックを使用すると、顧客は選択したオプションで商品を直接カートに追加できます。

WooCommerce 10.0で導入された新しい「カートに追加 + オプション（ベータ版）」ブロックは、完全にブロック化されたカスタマイズ可能なカート追加フォームを提供します。このブロックは、単品商品テンプレートまたは単品商品ブロック内で使用でき、複雑な商品タイプや柔軟なデザインニーズに最適です。

新しいブロックには主な機能が備わっています:

- ページを再読み込みせずに商品をカートに追加できるため、よりスムーズでスピーディーな操作性を実現します。
- 数量の更新は、ミニカート、商品コレクション、そしてこのブロック全体に即座に反映されます。画面更新は不要です。
- ドロップダウンの代わりにピルビューを使用することで、より魅力的なバリエーション選択体験を実現できます。

デフォルトでは、このブロックには以下が含まれます。

- **カートに追加ボタン:** 設定した商品をカートに追加するためのボタンを表示します。このボタンのスタイル（塗りつぶしスタイルやアウトラインなど）をカスタマイズできます。
- **商品数量（ベータ版）:** 商品をカートに追加する前に、顧客が数量を入力できるフィールドです。

製品タイプに応じて、追加の選択ブロックが利用可能です。

- **バリエーションセレクター：属性名（ベータ版）：** 属性名の表示方法をカスタマイズします（例：「色」、「サイズ」）。ブロック設定から色、サイズ、タイポグラフィを設定できます。
- **バリエーションセレクター：属性オプション（ベータ版）：** 商品の属性値をピル形式またはドロップダウン形式で表示できます。

#### 動的バリエーションの更新

WooCommerce v10.3.0 では、「カートに追加 + オプション」ブロックが[更新](https://developer.woocommerce.com/2025/10/22/woocommerce-10-3-cogs-comes-to-core-and-mcp-beta/)され、バリエーションが選択されると、関連する製品ブロックが自動的に更新され、正しいバリエーション データが表示されるようになりました。

例えば、バリエーションを選択すると、商品画像が更新され、選択したバリエーションの商品画像が表示されます。同様に、「カートに追加 + オプション」ブロックでは、バリエーションが個別に販売されている場合、「商品数量」ブロックも非表示になります。

- **グループ化された製品セレクター（ベータ版）:** 顧客がまとめて選択してカートに追加できる、構成された製品のグループを表示します。
- 構成されたグループ化された製品、製品名、製品価格、在庫を入力するためのセレクターを備えたグリッド形式で表示されます。
