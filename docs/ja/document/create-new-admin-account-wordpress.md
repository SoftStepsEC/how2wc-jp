title: WordPressで新しい管理者アカウントを作成する

より複雑なケースにおいて迅速なサポートを提供するため、ハピネスエンジニアがお客様のサイトの管理者アカウントをリクエストし、問題を詳しく調査させていただく場合がございます。その場合、一時的な管理者ログインの作成をお願いする場合がございます。このログインは、サポート完了後、**削除**していただきますようお願いいたします。

**注記：**

GDPR規則を遵守するには、Wooのような第三者へのアクセス権の共有について利用規約で開示する必要があります。[GDPRに基づくストアオーナーの責任](https://woocommerce.com/posts/the-gdpr-and-you-the-woocommerce-store-owner/)については、当社のブログで詳しくご紹介しています。

1. WordPress管理ダッシュボードで、**「ユーザー」>「新規追加」**に移動します。
2. **ユーザー名（必須）**欄に「woologin」と入力します。
3. **メールアドレス（必須）**欄に「woologin@woocommerce.com」と入力します。
- ハピネスエンジニアは、パスワードをリセットする必要がある場合、このメールアドレスにアクセスできます。
  
4. WordPressによって生成された**パスワード**をコピーし、[QuickForget.com](http://quickforget.com/) から送信してください。
5. **役割** が「管理者」に設定されていることを確認してください。
6. サイトが**英語**以外の言語で表示されている場合、このユーザーの言語を「英語」に設定してください。
7. **新しいユーザーを追加** を選択します。

管理者ユーザーを作成したら、これらの情報を安全に送信する方法は次のとおりです。[QuickForget.com](https://quickforget.com/) にアクセスし、「**秘密は…**」フィールドに以下の情報を入力してください。

- ウェブサイトのログインURL
- ユーザー用に作成したパスワード

問題の修正と解決が完了後、アカウントが不要になりましたら、サイトからアカウントを削除してください。削除するには、**「ユーザー」>「すべてのユーザー」** に移動し、woologin アカウントにマウスオーバーして **削除** を選択してください。

繰り返しになりますが、セキュリティ上の理由から、Woo Happiness サポートとのやり取りが終了したらすぐにこれを実行することをお勧めします。

[WordPress の創設者は、ユーザー名として `admin` を **絶対に** 使用すべきでない理由について次のように述べています](http://ma.tt/2013/04/passwords-and-brute-force/)。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
