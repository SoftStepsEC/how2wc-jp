PHP は最も人気のあるサーバー側スクリプト言語であり、ほとんどの WooCommerce ストアを含む何百万もの Web サイトで使用されています。

WooCommerceストアの通知からこのページにアクセスした場合、ストアでは**古くメンテナンスされていない**バージョンのPHPが稼働しています。ウェブサイトのパフォーマンスが（大幅に）低下するだけでなく、期待通りに動作せず、セキュリティ上の脆弱性が生じる可能性があります。

PHPは、WooCommerceウェブショップを支えるスクリプト言語です。他のソフトウェアと同様に、PHPもセキュリティ問題の修正や機能改善のために定期的にアップデートされます。他のソフトウェアと同様に、PHPのバージョンを常に最新の状態に保つことが重要です。

ほとんどの場合、PHPのバージョンを自分で更新することはできず、ホスティング会社に連絡する必要があります。アップグレードは簡単なプロセスで、ホスティング会社がウェブサイトに影響を与えたり、料金を請求したりすることなく、お客様に代わって実行してくれるはずです。ホスティング会社に送付できる手紙は以下の通りです。

> ホスト様
> 貴社のサーバーで WooCommerce ウェブショップを運営しているのですが、WooCommerce では PHP 7.0 以上の使用を推奨しています。WooCommerce が使用しているコンテンツ管理システムである WordPress の要件ページ（https://wordpress.org/about/requirements/）では、推奨バージョンとして PHP 7.4 が記載されています。
> 私のホスティングサービスが PHP 7.0 以上をサポートしているかどうか、またアップグレード方法について教えていただけますか？
> ご返信をお待ちしております。

VPS サーバーをお持ちの場合は、[PHP 5 からアップグレードする方法](https://laracasts.com/discuss/channels/tips/how-to-upgrade-from-php-5-to-php-70-on-ubuntu-1404-with-nginx-146) を参照してください。

ご利用のホスティング会社がPHP 7.0以上をサポートしていない場合は、PHP 7.0以上をサポートするホスティング会社を探すことをお勧めします。[おすすめのWordPressホスティングソリューション](https://woocommerce.com/hosting-solutions/)のリストをご用意しており、すべてPHP 7.0以上をサポートしています。他のホスティング会社にお問い合わせいただく場合は、購入前にウェブサイトがどのPHPバージョンで動作するか必ずご確認ください。

PHPのウェブサイトには、[サポート対象外のバージョン一覧（日付を含む）](http://php.net/eol.php)が掲載されています。WordPressプラグインをご自身で開発される場合は、[WPupdatePHP](http://www.wpupdatephp.com/)というPHPライブラリをご検討ください。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
