より複雑な税金設定もありますが、WooCommerce における税金の仕組みを理解しておくと、設定やトラブルシューティングに役立ちます。いくつかのニュアンスを見ていきましょう。

追加の参考資料として関連ガイド:

- [WooCommerce で特定の税金を設定する方法](https://woocommerce.com/document/configuring-specific-tax-setups-in-woocommerce/)
- [デジタル製品に対する EU VAT 税率の設定](https://woocommerce.com/document/setting-up-eu-vat-rates-for-digital-products/)

**製品価格** は次のいずれかとして定義できます。

1. 税込み
2. 税抜

**税込みで定義**する場合、ストアの**拠点国**の税金を指します。例えば、ストアの拠点国がイギリスの場合、商品価格は税率20%で定義します（**送料**は常に**税抜きで定義**されます）。

価格に税金が含まれている場合、顧客の所在国の税率がストアの拠点国と異なる場合、顧客に表示される価格はその国に基づいて変わる可能性があることに留意することが重要です。

たとえば、税率が 1 つだけ設定されている英国ベースのストアでは、エクスペリエンスは次のようになります。

- 税込価格120ポンドの商品を作成します。
- 英国在住のお客様は120ポンド（100ポンド + VAT 20ポンド）をお支払いいただきます。
- 英国以外のお客様は、居住地に応じた税率がないため、100ポンドをお支払いいただきます。

**または**

英国を拠点とするストアが英国 (20%)、フランス (19%)、ノルウェー (25%) に販売している場合、ストアは次のように機能します。

- 税込価格120ポンドの商品を作成します。
- 英国のお客様は120ポンド（100ポンド + VAT 20ポンド）をお支払いいただきます。
- フランスのお客様は119ポンド（100ポンド + 消費税 19ポンド）をお支払いいただきます。
- ノルウェーのお客様は125ポンド（100ポンド + 消費税 25ポンド）をお支払いいただきます。
- 英国、フランス、ノルウェー以外の国では、税率が適用されないため、100ポンドをお支払いいただきます。

[複数の国を対象としたシナリオを設定する方法の実用的な例はこちらです](https://woocommerce.com/document/configuring-specific-tax-setups-in-woocommerce/#tax-only-certain-countries)。

販売者によっては、場所や税金に関わらず同じ価格を請求したい場合があります。その場合、税金の差異を補うために基本価格が動的に調整されます。[この機能を有効にする方法については、こちらのリンクをご覧ください](https://woocommerce.com/document/configuring-specific-tax-setups-in-woocommerce/#same-price-regardless-of-location)。

税金の計算は、商品ごとではなく**行ごと**に行われます。以下の計算式を使用して、特定の価格に適用される税金額を算出できます。

**税抜き**の商品価格に対して税率 20% を計算するには:

```
Tax = Line Price * 0.2
```

**税込**の商品価格に対して税率 20% を計算するには:

```
Tax = Line Price - ( Line Price / 1.2 )
```

価格に既に税金が含まれている場合、税金は合計明細価格のちょうど20%にならないことに注意してください。前のセクションの例では、合計明細価格（120ポンド）の20%を単純に計算すると、**誤った24ポンド**となり、ベース商品の20%と既に加算されている税金が含まれます。そのため、計算方法が異なります。

税抜き価格の場合、各国の税金を計算するのは簡単です。**税抜き価格に税率を掛けるだけです。**税込み価格の場合は少し複雑です。

英国（税率20%）からドイツに販売する例を見てみましょう。商品の英国での価格は税込み120です。英国を拠点とする販売者がドイツで商品を販売する際に税金を課す必要がない場合、ドイツでの販売価格は以下のようになります。

- 英国価格 = 120 / 1.2 = 100
- 英国税 = 20
- ドイツ価格 = 100

上記と同じ例ですが、19%の税率が適用されるデジタル商品を考えてみましょう。まず英国の税率を差し引き、次にドイツの税率を次のように適用します。

- 英国価格 = 120 / 1.2 = 100
- 英国税 = 20
- ドイツ価格 = 100 * 1.19 = 119

よくある機能リクエストは、どこでも同じ料金を請求し、税金の差額を吸収することです。前の2つの例に基づくと、結果は次のようになります。

英国（税率20%）からドイツへ販売する場合、商品の価格は税込み120です。英国を拠点とする販売者がドイツで物理的な商品に税金を課す必要がない場合、以下のようになります。

- 英国価格 = 120
- 英国税 = 20
- ドイツ価格 = 120
- ドイツ税 = 0

ドイツの価格はイギリスの価格と同じですが、ドイツには税金が含まれていないため、基本製品価格が高くなります。

デジタル商品の場合、19% のドイツの税金が課せられる必要があり、計算は次のようになります。

- 英国価格 = 120
- 英国税 = 20
- ドイツ価格 = 120 / 1.19 = 100.84
- ドイツ税 = 120 - (120 / 1.19) = 19.1596

ドイツでの価格は同じですが、税金を徴収しているため、基本製品価格は英国よりも少しだけ高くなっています。
