title: 単一注文ページと手動での注文の追加

WooCommerce の個別注文ページでは、ストアの顧客が行った注文の詳細を確認できます。管理者やショップマネージャーは、このページから注文を追加したり編集したりすることもできます。

このドキュメントでは次の内容を説明します。

- 単一注文ページに表示される情報。
- 注文の編集方法。
- さまざまな種類の[注文メモ](https://woocommerce.com/document/managing-orders/view-edit-or-add-an-order/#order-notes)とその追加方法。
- [手動で注文を追加する](https://woocommerce.com/document/managing-orders/view-edit-or-add-an-order/#manually-adding-an-order)方法。

**単一注文**ページから、注文データの表示、編集、更新を行うことができます。いくつかのオプションは以下のとおりです。

- 注文ステータスを変更します。
- 商品、数量、価格、税金の調整など、注文項目を編集します。
- 注文の詳細を顧客にメールで送信したり、ダウンロード権限を再発行したりするなど、注文に関する操作を実行します。
- 商品の **メタ** を変更し、メタ情報を削除または追加することで、商品のバリエーションを編集します。
- クーポンを適用します。
- 注文に適用するクーポンコードが必要です。
- クーポンの使用回数は追跡され、注文からクーポンを削除することもできます。
- クーポンを有効にするには、注文が未払いである必要があります。
  
- 手数料を追加します。金額または割合を入力して、注文に手数料を追加できます。
- 手数料をマイナスにすると、税金は他のすべての商品に配分され、カートの合計金額がゼロを下回ることがありません。
  

**注:** 商品メタデータ、手数料、クーポン、配送料などの注文詳細の変更は、「**保留中**」または「**保留中**」ステータスの既存の注文に対してのみ可能です。注文が「**処理中**」状態、または[注文ステータスフロー](https://woocommerce.com/document/managing-orders/order-statuses/#visual-diagram-to-illustrate-order-status-flow)のさらに先に進むと、編集できなくなります。

注文詳細パネルには次の内容が含まれます。

- 注文番号
- お支払い情報
- 注文日時
- 注文状況
- 顧客情報
- ユーザー名とメールアドレス、および顧客のプロフィールや過去の購入履歴を表示するためのリンク
- 請求情報
- 配送情報
  

このセクションの詳細のほとんどは更新および/または変更できます。

- 日時を変更するには、ドロップダウンの日付セレクターと、時間に対応する数量セレクターを使用します。
- ステータスを変更するには、ドロップダウンで適切なステータスを選択します。
- 顧客を変更するには、現在の顧客を選択し、新しい顧客を検索します。

「請求」と「配送」の項目では、他にもいくつかの詳細を変更できます。変更するには、それぞれの横にある鉛筆アイコンを選択してください。

- 「請求」では、以下の項目を変更できます。
- 請求先住所 - 「請求先住所を読み込む」を選択して、顧客プロフィールから読み込むこともできます。
- メールアドレス
- 電話番号
- お支払い方法と詳細
  
- 「配送」では、以下の項目を変更できます。
- 配送先住所 - 顧客プロフィールから読み込むか、請求先住所からコピーすることもできます。
- 顧客が入力したメモ
  

必要な変更を行ったら、[**注文アクション**] メタボックスの [**更新**] ボタンを押して注文を更新します。

注文ページの次のパネルは**注文商品**パネルです。ここには、商品、配送の詳細、注文概要が表示されます。

- 各商品項目の行には、以下の情報が表示されます。
- 商品画像
- 商品名
- 単価
- 数量
- 合計（割引考慮後の価格 × 数量）
- 税金
  
- その下には配送の詳細が表示されます。これには以下の情報が含まれます。
- 配送方法
- 梱包商品
- 合計金額
- 税金
  
- 最後のセクションには、注文費用の概要が表示されます。注文が返金されると、このセクションの内容は変更されます。デフォルトでは、以下の内容が表示されます。
- 商品小計：税抜価格
- クーポン：クーポン利用に基づいて差し引かれた金額。使用されたクーポンはこのセクションの左側に表示されます。
- 送料：注文の送料
- 税金：注文全体にかかる税金の額。注文に適用された税コードに置き換えられます。
- 注文合計：上記の費用の合計
- 支払金額と決済ゲートウェイで徴収される手数料の概要
  

**注:**WooCommerceでは、各注文に対して1つの支払い方法しか受け付けられません。注文が既に支払われており、顧客が追加の商品をご希望の場合は、既存の注文に新しい商品を追加するのではなく、[新しい注文を作成](https://woocommerce.com/document/managing-orders/view-edit-or-add-an-order/#manually-adding-an-order)してください。その後、顧客に[支払いリンクを記載した請求書](https://woocommerce.com/document/managing-orders/paying-for-orders/#pay-for-order-email-link)を送信できます。

払い戻しを除き、注文ステータスが「**支払い保留中**」または「**保留中**」でない限り、注文項目を編集することはできません。

製品項目を編集するには、編集する製品ラインの横にある鉛筆アイコンを選択します。

編集できる製品項目は次のとおりです。

- **メタ情報を追加**: メタ情報を追加または削除して、商品変数のオプションを変更します。
- **数量**: 顧客が購入する商品の数。
- **合計**: 税引前割引**適用前の明細価格と税額。
- **税金**: 税金。例えば、顧客が免税対象の場合、税金を削除することができます。

**その他のアクション** このウィンドウの下部にはさらに 4 つのアクションがあります。

- 「商品を追加」には、6つの新しいオプションが表示されます。
- 商品を追加：注文に商品を追加します。
- 手数料を追加：ギフトラッピングなどの追加料金を追加します。
- 送料を追加：送料を追加します。追加が完了したら、鉛筆アイコンを選択して、名前、配送方法、送料、税金を更新します。
- 税金を追加：注文の各セクションに税金コードを追加します。
- キャンセル：変更を行わない場合はキャンセルします。
- 保存：変更を加えたら保存します。
  

- **クーポンを適用**：お客様がクーポンの追加を忘れた場合、または支払い前にお客様に特典を与えたい場合は、このオプションを選択すると、クーポンコードを適用できるモーダルが表示されます。
- **返金**：お客様に返金します。手動返金と自動返金の詳細については、[WooCommerce 返金](https://woocommerce.com/document/woocommerce-refunds/) をご覧ください。
- **再計算**：注文商品に変更を加えた後、ストア設定に基づいて自動的に新しい税金計算を実行したい場合は、「再計算」を選択できます。これは、商品、クーポン、配送方法などを追加または削除する場合に便利です。ストアの税金設定はお客様の住所に基づいて適用されるため、手動で行った税金の変更は削除されることにご注意ください。

カスタムメタフィールドを追加するには、**カスタムフィールド** メタボックスを使用します。これは [WordPress のコア機能](https://wordpress.org/documentation/article/assign-custom-fields/) であり、WooCommerce で注文に追加データを含める必要がある場合に非常に便利です。

**注文メモ**パネルには注文に添付されたメモが表示され、支払い結果や在庫数の減少などのイベントの詳細を保存したり、顧客が確認できるように注文にメモを追加したりするために使用できます。一部の決済ゲートウェイでは、デバッグ用のメモも追加されます。

注文には次のメモ タイプが追加されます。

- **システムメモ** (紫色): WooCommerce または拡張機能によって自動的に追加されるメモで、支払いの成否などの詳細が含まれます。システムプロセス (チェックアウトなど) の結果として注文ステータスが変更されると、メモは紫色で表示されます。システムメモは顧客には表示されません。ただし、注文ステータスの変更によっては、ストア管理者または顧客に [通知](https://woocommerce.com/document/configuring-woocommerce-settings/emails/#email-notifications) が送信されます。
- **一般メモ** (灰色): ストア管理者または拡張機能によって社内参照用に追加されます。これには、管理者による手動の注文ステータスの変更や、非公開メモが含まれます。顧客にはこれらのメモは表示されませんが、通知を受け取る場合があります (例: ショップマネージャーが注文ステータスを「処理中」から「完了」に変更すると、[メール設定](https://woocommerce.com/document/configuring-woocommerce-settings/#email-settings) に応じてメールが送信される場合があります)。
- **顧客メモ** (青): 顧客のために注文に手動で追加されたメモです。メモはメールで届き、「マイアカウント」エリアで注文の詳細を表示すると表示されます。ただし、顧客メモに関するメール通知は、メール設定で無効にすることができます。

メモは、顧客や他の店舗マネージャーとのコミュニケーションに非常に役立つツールです。配送の追跡番号を追加したい場合や、在庫の遅延など、顧客メモを追加すると、自動的に通知されます。

メモを追加するには:

1. テキストエリアにメモの**内容**を追加します。(ハイパーリンク用の<a>タグなど、一部のHTMLは使用できます。)
2. ドロップダウンから**プライベートメモ**または**顧客へのメモ**を選択します。
3. **追加**ボタンを押します。

管理者またはショップマネージャーのアクションの結果として作成されたメモは灰色または青で表示され、タイムスタンプの後に「by {username}」と表示されます。

注文を追加するには:

1. **WooCommerce > 注文** に移動します。
2. ページ上部の **注文を追加** を選択します。単一注文ページが表示されます。
3. 顧客情報を**入力** し、明細項目を追加、クーポンを適用、手数料を適用、合計金額を計算します。これらは [注文項目の編集または追加](https://woocommerce.com/document/managing-orders/view-edit-or-add-an-order/#editing-or-adding-order-items) と同じです。
4. 新しい注文の **ステータス** を設定します。例：支払いが必要な場合は、「支払い保留中」を使用します。
5. **保存** をクリックします。

「注文アクション」ドロップダウンを使用して、注文の詳細と支払い手順を顧客に電子メールで送信します。
