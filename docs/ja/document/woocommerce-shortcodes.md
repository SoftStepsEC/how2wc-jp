**注記：**

WooCommerceブロックは、投稿やページに商品を表示するための最もシンプルかつ柔軟な方法です。[WooCommerceブロックの詳細](https://woocommerce.com/document/woocommerce-store-editing/blocks/)

WooCommerceは従来、WordPressページに動的なコンテンツや機能を追加するためにショートコードを利用してきました。WordPressがGutenbergブロックベースエディターへと進化するにつれ、その焦点はショートコードから、よりインタラクティブで視覚的に統合されたブロックへと移りました。

ブロックエディターはインタラクティブにカスタマイズ可能なブロックと強化された編集機能を提供していますが、多くのストアが依然としてショートコードに大きく依存していることを認識しています。[ショートコードブロック](https://wordpress.org/documentation/article/shortcode-block/)内では、従来のショートコードを使用できます。

ショートコードはWordPressの投稿やページで使用できます。ブロックエディターを使用している場合は、**ショートコードを**ショートコードブロックに貼り付けることができます。

クラシック エディターを使用している場合は、ショートコードをページまたは投稿コンテンツに直接貼り付けることができます。

一部のショートコードは、より具体的な内容にしたり、特定の方法でコンテンツを表示したりするために変更できます。ショートコードに**属性**（引数またはargsとも呼ばれます）を追加することで、ショートコードを変更することができます。

たとえば、`[add_to_cart]` ショートコードに属性 `id` を追加すると、特定の商品の「カートに追加」ボタンが作成されます。`[add_to_cart id=”99”]` は、ショップ内の ID が `99` の商品の「カートに追加」ボタンを表示します。

WooCommerce は [ブロックベースのストア編集](https://woocommerce.com/document/woocommerce-store-editing/) に移行したため、各ショートコードのブロック代替へのリンクも追加しました (該当する場合)。

**注記**;

様々なブロックが、様々なオプションで商品を表示します。例えば、「全商品」ブロック、「厳選商品」ブロック、「カテゴリー別商品」ブロック、「タグ別商品」ブロックなどです。これらのブロックは廃止され、より堅牢で汎用性の高い[商品コレクションブロック](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/)に置き換えられます。

サポートが必要な場合は、[ショートコードとコード スニペットのトラブルシューティング](https://woocommerce.com/document/woocommerce-shortcodes/troubleshooting-shortcodes/)を参照して、ショートコード関連の一般的な問題に関するサポートを受けてください。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
