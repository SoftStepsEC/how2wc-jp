WooCommerce コアプラグインの WooCommerce.com Marketplace の提案設定により、ストア所有者は受け取る推奨事項や提案の種類を管理して、エクスペリエンスを柔軟にカスタマイズできます。

このドキュメントでは、利用可能な設定の概要、それらの動作、カスタマイズのベスト プラクティス (プログラムで提案を無効にするフィルターを含む) について説明します。

マーケットプレイスの提案は、ストアを強化できる拡張機能、サービス、機能に関するパーソナライズされた推奨事項を提供することで、ストアのオーナーを支援するように設計されています。

関連性と価値を保証するために、これらの提案はストアの現在の設定、インストールされているプラ​​グイン/拡張機能、および使用パターンに基づいています。

この設定を有効にすると、WooCommerce.com はストアの WP 管理ダッシュボードの関連領域でパーソナライズされた拡張機能の推奨事項を表示できるようになります。

これらの提案は、成長を促進し、ストアの効率を高めるのに役立つツールや機能を見つけるのに役立つように設計されています。

マーケットプレイスの提案設定にアクセスして構成するには:

1. ストアのWP Adminダッシュボードで、*WooCommerce > 設定* に移動し、**詳細設定** タブをクリックします。
2. 次に、タブ付きナビゲーションの下のサブメニューにある **WooCommerce.com** リンクをクリックします。
3. 「マーケットプレイスの提案」の下に、「提案を表示」チェックボックスがあります。
- チェックを入れると、WooCommerce はストアのWP Adminダッシュボードにパーソナライズされた提案を表示します。
- チェックを外すと、WooCommerce.com からの提案はパーソナライズされません。代わりに、拡張機能の静的リストが表示されます。
  
4. 選択したら、[**変更を保存**] ボタンをクリックします。

フィルターを使用して、プログラム的にマーケットプレイスの提案を無効にすることができます。手順については、[開発者向けドキュメント](https://developer.woocommerce.com/docs/disabling-marketplace-suggestions-programmatically/)をご覧ください。

- **エクスペリエンスをカスタマイズ:** 設定をカスタマイズして、ストアに最も関連性の高いおすすめが表示されるようにしましょう。
- **おすすめを確認:** 表示される提案をじっくりと検討しましょう。これらの提案は、ストアの機能と顧客体験を向上させるための貴重なインサイトやソリューションを提供します。

マーケットプレイスの提案の設定で問題が発生した場合:

- WooCommerce と WordPress の両方のバージョンが最新であることを確認してください。
- ストアの WooCommerce.com への接続を確認し、更新情報や推奨事項を確認してください。
- 問題が解決しない場合は、[公式 WooCommerce サポートフォーラム](https://wordpress.org/support/plugin/woocommerce) にアクセスするか、[サポートに問い合わせ](https://woocommerce.com/my-account/contact-support/) してさらにサポートを受けてください。

まだ質問があり、サポートが必要ですか?

このドキュメントは、無料の[コア WooCommerce プラグイン](https://wordpress.org/plugins/woocommerce/)に関するものです。このプラグインのサポートは、[WordPress.org のコミュニティフォーラム](https://wordpress.org/support/plugin/woocommerce) で提供されています。このフォーラムを検索すると、同じ質問が過去に投稿され、回答されているケースがよくあります。フォーラムを利用するための WordPress.org アカウントをまだ作成していない場合は、[作成方法はこちら](https://make.wordpress.org/contribute/join/)をご覧ください。

- ここで紹介したコア機能を**拡張**したい場合は、[WooCommerce マーケットプレイス](https://woocommerce.com/products/)で利用可能な拡張機能をご確認ください。
- 継続的な高度なサポートや WooCommerce 向けの**カスタマイズ**が必要ですか？[Woo エージェンシー パートナー](https://woocommerce.com/customizations/)をご活用ください。
- 独自の WooCommerce 統合機能や拡張機能を開発している**開発者**ですか？[開発者向けリソース](https://developer.woocommerce.com/)をご確認ください。
