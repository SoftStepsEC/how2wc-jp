## ドキュメント

- [サイトエディターの使い方](https://woocommerce.com/document/woocommerce-store-editing/the-editor/) サイトエディターは、テンプレートとブロックを使用してサイトをカスタマイズする場所です。このページでは、ストアのテンプレートとページコンテンツを編集するための中心的なツールであるWordPressサイトエディターの概要を説明します。エディターへのアクセス方法、リストビューでのブロックとコンテンツの操作方法、やり直しや[…]の方法を学びます。
- [テンプレート](https://woocommerce.com/document/woocommerce-store-editing/templates/) テンプレートは、複数の投稿、ページ、商品の外観とレイアウトを決定するブロックの配置です。カスタムテンプレートを作成することも、テーマやプラグインに付属のテンプレートを使用することもできます。テンプレートに変更を加えると、エディターはそのテンプレートを使用しているすべてのページ/投稿のブロックを更新します。 […] について詳しくはこちらをご覧ください。
- [パターンとテンプレートパーツ](https://woocommerce.com/document/woocommerce-store-editing/patterns-template-parts/) WordPress とブロックエディターでは、パターンとテンプレートパーツを使用すると、オンラインストアのレイアウトとデザインをカスタマイズでき、同じデザインを何度も作り直す手間を省くことができます。パターンは、ページや投稿に簡単に挿入できる、あらかじめデザインされたセクションです。パターンは同期することも、独立して（同期せずに）使用することもできます。[…]
- [WooCommerce でのブロックの使用](https://woocommerce.com/document/woocommerce-store-editing/blocks/) WooCommerce には、投稿、ページ、サイトエディターのパターンやテンプレートに挿入できるブロックが多数用意されており、カタログのカスタマイズ、商品のプロモーション、そして買い物客に快適なチェックアウト体験を提供できます。このガイドでは、ブロックの基本的な仕組み、設定の確認方法、そして[…] を元に戻す方法について説明します。
- [WooCommerce用ミニカート](https://woocommerce.com/document/mini-cart-for-woocommerce/) ショートコードを使って、ショップページやWooCommerceカートにWooCommerceミニカートを追加し、顧客がカートの詳細に素早く簡単にアクセスできるようにします。設定と構成 プラグインの設定にアクセスするには、「WooCommerce」>「ミニカート」>「一般設定」に移動します。サイドカートの設定 このセクションでは、[…] について説明します。
- [アコーディオングループブロック](https://woocommerce.com/document/accordion-group-block/) アコーディオングループブロックは、ストアページや商品ページで折りたたみ可能なコンテンツを、すっきりとインタラクティブに表示する方法を提供します。このブロックは、情報を管理しやすく展開可能なセクションに整理し、読みやすさとユーザーエクスペリエンスを向上させるように設計されています。このブロックは、複数のアコーディオンアイテムをグループ化することで、情報をコンパクトでユーザーフレンドリーな形式で提示できます。[…]
- [商品コレクションブロック](https://woocommerce.com/document/woocommerce-store-editing/customizing-shop-page-catalog/product-collection-block/) 商品コレクションブロックは、商品アーカイブの表示とカスタマイズ、そして商品のおすすめ表示を支援するために設計されています。ブロックを挿入すると、現在編集中のテンプレートまたはページに応じて、独自のカスタム商品コレクションを作成するか、バリエーション/レシピのリストから選択するかを選択するように求められます。[…]

- [商品ギャラリーブロック](https://woocommerce.com/document/product-gallery-block/) 商品ギャラリーブロックは、ストア内で商品画像やメディアを柔軟かつインタラクティブに表示できるブロックです。このブロックでは、サムネイル画像とギャラリービューの拡大画像を組み合わせることで、商品の詳細なビジュアル表現をお客様にご覧いただけます。ブロック構造は、2つの主要な内部ブロックで構成されています。[…]
