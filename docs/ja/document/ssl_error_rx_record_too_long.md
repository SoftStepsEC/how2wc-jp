このエラー:

> SSL は最大許容長を超えるレコードを受信しました。(エラーコード: ssl_error_rx_record_too_long)

SSL/Apacheの設定ミス（通常はポート設定とVirtualHost設定）が原因です。ほとんどの場合、ホスティングプロバイダーまたはサーバー管理者が問題を修正する必要があります（WooCommerceの問題ではありません）。

SSL が必要ない場合は、**WooCommerce > 設定** で「SSL を強制」設定をオフにすることもできます。
