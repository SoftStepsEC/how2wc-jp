当社のプレミアム配送方法の多く、特に運送業者の API 経由で配送料を計算する方法には、購入するすべての商品に最適な箱または封筒を決定する Box Packers が組み込まれています。

**Box Packersの仕組み:**

梱包担当者は、利用可能な箱／封筒の寸法（指定されている場合は「**内寸**」を使用）とカート内のすべての商品の寸法を確認し、注文を**最小限の箱数**で梱包するよう努めます。1つの箱にすべての商品が収まらない場合は、まずそれらの商品が収まる最大の箱で梱包し、その後、追加の箱を使用して残りの商品を梱包します。

**梱包担当者は、寸法に加えて、購入商品の重量** **と、利用可能な箱/封筒の最大重量** **も考慮します。これにより、より正確な梱包が可能になり、箱の重量超過を防ぐことができます。最大重量を超える荷物は詰め込まないためです。ご注文の商品が1つの箱/封筒に収まらないほど重い場合は、複数の梱包を使用します。

一部の配送拡張機能では、2 つの異なる **Box Packer ライブラリ** から選択できます。

- **スピードパッカー：** スピードパッカーは主に**容積ベース**です。商品の寸法が箱/封筒の寸法以下であることを確認した後、このボックスパッカーは商品の容積（長さ x 幅 x 高さ）を計算し、それをパッケージの**容積容量**と比較して最適なサイズを見つけます。このボックスパッカーは数学的計算を使用するため、特に数量が多い場合は計算が高速化し、使用するリソースも少なくて済みます。ただし、数学的計算は実際の箱詰めと必ずしも一致するとは限らないため、注文を正しく梱包するには商品またはパッケージの寸法に若干の調整が必要になる場合があります。
- **アキュレートパッカー：**アキュレートパッカーは主に**回転ベース**です。商品の寸法が箱/封筒の寸法以下であることを確認した後、このボックスパッカーはビデオゲームの「テトリス」のように、各商品をパッケージに追加する前に複数回回転させます。数量が多い場合、これらの計算はより集中的になり、スピード パッカーほど速くはありませんが、人間が箱を梱包する方法と非常に似ているため、通常はより正確です。

すべての計算ツールで最も重要な部分は、個々の商品ページ、特に「配送」タブです。そこでは、WooCommerce設定で選択した単位で、重量、長さ、幅、高さが表示されます。

この欄は商品の**発送**寸法を入力する場所であり、実際の梱包時の寸法ではないことにご注意ください。商品または梱包の発送時のサイズは、余裕を持たせる余裕や余裕のあるサイズも含めて、ここで入力するサイズとなります。

例えば、映画ポスターのワンシートを販売するとします。ポスターは、レイアウトすると幅27インチ、高さ41インチ、厚さ0.01インチになります。しかし、これらの寸法を入力すると、少なくとも27.01インチ×41.01インチ×厚さ0.02インチの箱が必要になり、無駄なスペースが発生し、料金が大幅に上昇してしまうでしょう。

代わりに、ポスターを丸めて筒に入れることになるでしょう。ワンシートは、一番細い辺が高さになるように巻かれています。つまり、約27インチ×3インチ×3インチのパッケージに巻き込まれます。円形になりますが、これらの寸法を測っておくことで、可能な限りタイトで合理的なパッケージに収まるようになります。

ほとんどの方法では、選択した方法に応じてデフォルトのボックスサイズが使用されます。これらのサイズは以下のウェブサイトで確認できます。

- [USPS](https://store.usps.com/store/browse/subcategory.jsp?categoryId=shipping-boxes)
- [UPS](https://www.theupsstore.com/pack-ship/moving-boxes-supplies)
- [FedEx](http://www.fedex.com/us/office/packandship/packing-services.html)
- [カナダ郵便](https://www.canadapost.ca/web/en/products/details.page?article=mailing_and_shipping)
- [オーストラリア郵便](http://auspost.com.au/parcels-mail/packaging-options.html)

一部の方法では、ご自身の箱や封筒をご使用いただくことも可能です。その場合、荷物の最大幅の外側寸法と内側寸法、そして最大重量を測る必要があります。これらの箱は、デフォルトの箱と併用されるものであり、完全に代替されるものではありませんのでご注意ください。

すべての配送方法には**デバッグモード**があります。このモードでは、配送業者の計算ツールとの間で送受信された情報と、配送に選択された箱が表示されます。出力は配送方法によって異なります。[配送方法とデバッグモードに関するドキュメント](https://woocommerce.com/document/online-shipping-calculators-debug-mode/)をご覧ください。
